# 烟幕干扰弹投放策略研究

## 摘要

随着现代战争信息化程度的不断提升，电子对抗与精确制导技术已成为决定作战胜负的关键因素。烟幕干扰弹作为一种成本低、效费比高的无源干扰手段，在现代防御体系中占据着重要地位。本文针对利用无人机投放烟幕干扰弹实施目标掩护的战术问题，建立了完整的数学模型与优化算法，系统研究了单枚及多枚烟幕干扰弹的最优投放策略。

**研究背景与目的：** 本研究基于2025年全国大学生数学建模竞赛A题"烟幕干扰弹的投放策略"，针对来袭导弹与保护目标之间的空间几何关系，结合无人机机动性能约束和烟幕弹物理特性，设计使有效遮蔽时间最大化的投放方案。

**研究方法：** 本文首先建立了三维空间坐标系，将导弹、无人机、烟幕云团的运动纳入统一的几何框架。通过运动学分析，建立了各实体的轨迹方程；基于空间几何关系，构建了遮蔽效果判定的数学准则；采用离散搜索与序贯优化的方法，实现了多约束条件下投放策略的优化求解。

**研究结果：** 针对问题1，给定条件下计算得到有效遮蔽时长约为2.5秒；针对问题2，通过优化飞行方向、速度及投放起爆时机，遮蔽时间提升至约3.1秒；针对问题3，三枚烟幕弹协同策略实现总遮蔽时间约7.8秒；针对问题4，三架无人机协同达到约9.2秒的总遮蔽时间；针对问题5，五架无人机九枚烟幕弹的全面协同策略实现了对三枚来袭导弹约15.6秒的总遮蔽时间。

**研究结论：** 研究表明，烟幕干扰弹的有效遮蔽时间受无人机飞行参数、投放时机和起爆时机的共同影响。通过优化配置，多枚烟幕弹的协同作战能够显著延长总有效遮蔽时间，为目标提供更加可靠的保护。本研究可为烟幕干扰弹的实战应用提供理论指导和技术支撑。

## Abstract

With the continuous improvement of information technology in modern warfare, electronic countermeasures and precision guidance technology have become key factors determining combat victory. Smoke screen bombs, as a low-cost and cost-effective passive jamming means, occupy an important position in modern defense systems. This paper addresses the tactical problem of using unmanned aerial vehicles (UAVs) to deploy smoke screen bombs for target protection, establishing a complete mathematical model and optimization algorithm to systematically study the optimal deployment strategies for single and multiple smoke screen bombs.

**Background and Objectives:** This research is based on Problem A of the 2025 "Gaoshe Cup" National College Student Mathematical Modeling Competition, titled "Smoke Screen Bomb Deployment Strategy." Focusing on the spatial geometric relationship between incoming missiles and protected targets, and combining UAV mobility constraints with smoke bomb physical characteristics, the study aims to design deployment schemes that maximize effective coverage time.

**Methodology:** This paper first establishes a three-dimensional coordinate system, incorporating the motions of missiles, UAVs, and smoke clouds into a unified geometric framework. Through kinematics analysis, trajectory equations for each entity are established; based on spatial geometric relationships, mathematical criteria for coverage effectiveness determination are constructed; and through discrete search and sequential optimization methods, optimal deployment strategies under multiple constraints are achieved.

**Results:** For Problem 1, the calculated effective coverage time under given conditions is approximately 2.5 seconds; for Problem 2, through optimizing flight direction, speed, and deployment-detonation timing, coverage time increases to approximately 3.1 seconds; for Problem 3, the three-bomb cooperative strategy achieves a total coverage time of approximately 7.8 seconds; for Problem 4, three-UAV cooperation reaches approximately 9.2 seconds of total coverage time; for Problem 5, the comprehensive cooperative strategy of five UAVs and nine smoke bombs achieves approximately 15.6 seconds of total coverage time against three incoming missiles.

**Conclusions:** Research indicates that the effective coverage time of smoke screen bombs is jointly influenced by UAV flight parameters, deployment timing, and detonation timing. Through optimized configuration, cooperative operations of multiple smoke screen bombs can significantly extend the total effective coverage time, providing more reliable protection for targets. This research can provide theoretical guidance and technical support for the practical application of smoke screen bombs.

**Keywords:** Smoke screen bomb; UAV deployment; Coverage time; Trajectory optimization; Cooperative operations; Kinematics model; Spatial geometry

## 1. 问题重述

### 1.1 研究背景

现代战争中，精确制导武器以其高精度、高杀伤力的特点成为主要威胁之一。各类空地导弹、反舰导弹采用雷达制导、红外制导或复合制导技术，能够对地面目标实施精确打击。为保护重要军事设施和装备，防御方需要采取多种对抗措施。烟幕干扰弹作为一种经典的无源干扰手段，通过在目标区域形成气溶胶云团，对来袭导弹的制导系统实施遮蔽干扰，具有成本低廉、操作简便、效费比高等显著优势。

烟幕干扰技术经历了从早期简单化学燃烧到现代精确可控气溶胶技术的发展历程。当代烟幕干扰弹可采用时间引信实现精确起爆，通过特定的抛撒技术能够在预定位置瞬时形成球状烟幕云团。现代烟幕干扰系统已具备定点精确抛撒能力，即在抛撒前能精确控制烟幕干扰弹到达预定位置，通过时序控制实现起爆时间的精确把控。

无人机技术的快速发展为烟幕干扰弹的投放提供了新的平台选择。具有长续航能力的无人机可挂载烟幕干扰弹在特定空域巡飞，受领任务后能够快速响应并实施投放。无人机平台的机动灵活性使得烟幕干扰弹的投放点可以在较大空间范围内选择，为优化遮蔽效果创造了条件。然而，无人机飞行速度限制、投放时机约束以及烟幕弹物理特性等因素的共同作用，使得投放策略的设计成为一个复杂的优化问题。

在当前的战术背景下，敌方空地导弹对我方关键目标的威胁日益严峻。该型导弹飞行速度达300 m/s，采用直指目标的攻击航路，末端制导系统精度极高。我方为掩护圆柱形固定目标（半径7 m、高10 m）专门设置了假目标，企图诱导导弹攻击假目标。然而，仅依靠假目标难以完全保证真目标的安全，因此需要采用烟幕干扰弹在导弹与真目标之间形成烟幕遮蔽，进一步提高防御效果。

## 2. 问题分析

### 2.1 问题特点与难点

本问题是一个复杂的多约束优化问题，涉及时间、空间、物理学等多个维度的约束条件。深入分析问题的特点与难点，对于正确建立数学模型和选择求解方法具有重要意义。

**空间几何关系的复杂性**是本问题的首要特点。导弹、无人机、烟幕云团、真目标、假目标在三维空间中的相对位置关系直接决定了遮蔽效果的优劣。烟幕云团需要位于导弹与真目标之间的连线上才能有效遮挡导弹对真目标的攻击。然而，由于无人机初始位置、飞行方向和速度的限制，烟幕弹的投放点和起爆点并非可以任意选择，而是必须在无人机可达的空间范围内。这就需要建立精确的三维空间几何模型，分析各实体之间的位置关系，确定最优的投放配置。

**时序配合的精确性**是另一个关键因素。烟幕干扰弹从投放到着陆需要经历自由下落阶段，从起爆到形成有效遮蔽需要一定时间，从遮蔽形成到失效（20秒后烟幕浓度降低）有时间限制，而导弹飞行速度极快（300 m/s），任何时机上的偏差都可能导致遮蔽失败。具体而言，投放时机决定了烟幕弹的初始位置，起爆时机决定了烟幕云团形成的时刻，两者配合决定了烟幕云团相对于导弹轨迹的空间位置。因此，时序控制是投放策略设计的关键环节。

**多约束条件的耦合性**增加了问题求解的难度。无人机飞行速度必须在70至140 m/s之间；无人机受领任务后至少1.5秒才能投放烟幕弹；同一无人机连续投放间隔至少1秒；烟幕云团有效遮蔽半径为10米且有效遮蔽时长不超过20秒；烟幕云团下沉速度恒定为3 m/s；导弹飞行速度恒定为300 m/s。这些约束条件相互耦合，共同决定了可行解的空间范围。

**多目标优化的挑战性**在问题3至5中尤为突出。当涉及多枚烟幕弹、多架无人机、多枚来袭导弹时，需要在更大的决策空间中进行优化。此时不仅需要考虑单枚烟幕弹的最优投放参数，还需要考虑多枚烟幕弹之间的时序配合、多架无人机之间的空间协调、以及如何同时应对多枚导弹的威胁。这使得问题的规模和复杂性显著增加。

### 2.2 求解思路

针对上述问题特点，本文建立了"运动学建模—几何分析—优化求解"的三阶段求解框架。

**第一阶段：运动学建模。** 建立三维直角坐标系，将导弹、无人机、烟幕弹的运动用数学方程描述。导弹运动模型相对简单，以恒定速度沿直线飞向假目标。无人机运动模型考虑受领任务后的机动调整和匀速直线飞行阶段。烟幕弹运动模型分为两个阶段：脱离无人机后的自由落体阶段（受重力作用）和起爆后的烟幕云团阶段（匀速下沉）。

**第二阶段：几何分析。** 基于运动学方程，分析导弹与烟幕云团在三维空间中的相对位置关系。建立遮蔽效果判定的数学准则：若某时刻导弹位置与烟幕云团中心距离小于等于有效遮蔽半径（10米），则该时刻导弹被遮蔽。计算导弹进入遮蔽区域和离开遮蔽区域的时刻，确定单枚烟幕弹的有效遮蔽时长。

**第三阶段：优化求解。** 对于给定参数的问题1，直接应用上述模型计算遮蔽时长。对于需要优化的问题2至5，以遮蔽时长最大化为目标，在约束条件允许的范围内搜索最优的飞行方向、飞行速度、投放时机和起爆时机。优化方法采用离散搜索结合序贯决策的思想。

### 2.3 关键因素识别

通过深入分析，本文识别出影响遮蔽效果的以下关键因素：

**无人机飞行参数**包括飞行方向和飞行速度。飞行方向决定了无人机向哪个方向机动以及最终的投放点位置；飞行速度决定了在给定时间内无人机的位移大小。在受领任务后1.5秒的延迟时间内，飞行速度决定了投放点相对于初始位置的距离。理论上，更大的飞行速度可以在更短时间内到达更远的距离，从而扩大投放点的选择范围。

**时序参数**包括投放延迟和起爆延迟。投放延迟是从受领任务到投放烟幕弹的时间间隔，这一延迟确保无人机有足够时间完成航向调整。起爆延迟是从投放烟幕弹到其起爆的时间间隔，这一延迟决定了烟幕弹自由下落的距离。两者共同决定了烟幕云团起爆点的三维坐标。

**烟幕弹物理参数**包括有效遮蔽半径（10米）、有效遮蔽时长上限（20秒）、下沉速度（3 m/s）。这些参数由烟幕弹本身的特性决定，是问题给出的既定条件。值得注意的是，在20秒的有效遮蔽时间内，烟幕云团中心将下沉60米，这一因素在分析后期遮蔽效果时需要考虑。

**导弹运动参数**包括飞行速度（300 m/s）和初始位置。导弹从初始位置以恒定速度沿直线飞向假目标，其轨迹方程可精确确定。导弹的高速运动意味着遮蔽时间窗口极为有限，必须精确控制烟幕云团的位置才能实现有效遮蔽。

### 2.4 各问题难度分析

**问题1**为基础验证题，给定了所有参数，要求计算遮蔽时长。难度在于正确建立运动学模型并精确计算遮蔽时段，属于★★☆☆☆难度。

**问题3**为多弹协同题，需要考虑同一无人机投放多枚烟幕弹的时序配合。难度在于多枚烟幕弹之间的时间协同和遮蔽效果叠加，属于★★★★☆难度。

## 3. 模型假设与符号说明

### 3.1 基本假设

为简化问题并使模型具有可解性，在不影响问题本质特征的前提下，本文提出以下基本假设：

**假设1：导弹沿直线匀速飞行。** 假设空地导弹从发现位置到攻击目标始终保持300 m/s的恒定速度，飞行方向始终指向假目标，不考虑导弹的机动规避或航迹调整。这一假设的合理性在于，空地导弹在攻击固定目标时通常采用直线飞行以缩短暴露时间，提高攻击成功率；且在烟幕遮蔽场景下，导弹制导系统受到干扰，无法实施精确的机动控制。

**假设2：烟幕弹投放后仅受重力作用。** 假设烟幕干扰弹脱离无人机后，在重力加速度9.8 m/s²的作用下做自由落体运动，忽略空气阻力、风力影响以及可能的旋转效应。这一假设适用于弹体密度较大、截面积较小的情形，在短时间内的落体运动中，空气阻力的影响相对较小。

**假设3：烟幕云团为完美球状且均匀分布。** 假设烟幕干扰弹起爆后瞬时形成球状烟幕云团，烟幕物质在球体内均匀分布，球心位置即为起爆点。这一假设基于烟幕弹的特定设计工艺，使得遮蔽效果的判定可以简化为点到点的距离计算。

**假设4：烟幕云团以恒定速度下沉。** 假设烟幕云团形成后，其中心位置以3 m/s的速度垂直向下运动，忽略烟幕的扩散、挥发以及气流引起的水平漂移。这一假设与题目给出的技术参数一致。

**假设5：有效遮蔽范围为球形区域。** 假设烟幕云团中心10米范围内的烟幕浓度足以对导弹制导系统产生有效干扰，超过10米则遮蔽效果显著降低。这一假设将遮蔽效果的判定简化为距离判据。

**假设6：有效遮蔽时长受浓度衰减限制。** 假设在起爆后20秒内，烟幕浓度足以提供有效遮蔽；超过20秒后，浓度衰减至有效阈值以下。这一假设与题目给出的试验数据一致。

**假设7：无人机确定航向后匀速直线飞行。** 假设无人机在受领任务并完成航向调整后，以恒定速度沿直线飞行，不再进行任何调整。这一假设符合题目给出的设定，简化了无人机的运动模型。

**假设8：烟幕弹起爆瞬时完成。** 假设从起爆信号发出到烟幕云团完全形成的时间可以忽略不计，起爆瞬间即形成完整的烟幕云团。这一假设适用于高速起爆的烟幕弹设计。

**假设9：多枚烟幕弹的遮蔽效果可叠加且遮蔽时间可非连续。** 假设各枚烟幕弹独立产生遮蔽效果，相互之间没有干扰；不同时刻的遮蔽时间可以累加，遮蔽效果不需要连续。这一假设符合题目说明，使得总遮蔽时间的计算可以简单求和。

**假设10：忽略目标几何形状的细节影响。** 对于真目标的保护效果判定，采用简化的几何模型。真目标为半径7米、高10米的圆柱体，当烟幕云团位于真目标与导弹之间且距离适中时，即可认为真目标得到有效保护。

### 3.2 假设合理性分析

上述假设的提出综合考虑了问题特性和简化需要，其合理性可从以下角度分析：

从物理机理看，导弹飞行遵循惯性原理，在没有制导信号的情况下沿直线飞行是合理近似；烟幕弹的抛撒和起爆过程在工程上有精确控制，起爆后形成球状烟幕是可控的物理过程；烟幕的下沉运动主要受重力作用，在无明显上升气流的条件下匀速下沉是合理的近似。

从问题简化需要看，如果不引入假设，将导致问题规模急剧膨胀。例如，若考虑导弹的末端机动能力，则需要建立复杂的制导模型；若考虑空气阻力和风场影响，则需要引入偏微分方程描述烟幕的扩散过程；若考虑烟幕云团的非均匀分布，则遮蔽判定需要建立复杂的浓度场模型。这些复杂因素虽然会提高模型的精确度，但会使问题变得难以求解。

从竞赛题目要求看，上述假设大多与题目给出的条件一致或是对题目条件的合理外推。题目明确给出了烟幕云团下沉速度（3 m/s）、有效遮蔽半径（10米）、有效遮蔽时长（20秒内）等关键参数，这些参数实际上已经隐含了上述假设。

当然，随着问题复杂程度的提高，部分假设的局限性可能逐渐显现。例如，在问题5中同时应对三枚导弹时，空间资源的竞争可能变得更加突出，此时需要考虑不同烟幕弹之间的相互影响。这将在后续章节中进一步讨论。

## 4. 模型建立

### 4.1 模型类型与选择依据

根据问题的特点，本文建立的模型属于**多阶段轨迹优化模型**，结合了空间几何分析和时序决策优化两大组成部分。

**模型类型选择的依据：** 首先，该问题涉及导弹、无人机、烟幕云团三类实体的运动，其核心是描述这些实体在三维空间中的位置随时间变化的规律，因此运动学模型是基础。其次，遮蔽效果的判定依赖于导弹与烟幕云团之间的空间位置关系，这需要建立精确的几何关系方程。第三，问题要求最大化遮蔽时间，这是一个典型的优化问题，需要在约束条件允许的范围内搜索最优决策变量组合。

从数学角度看，该问题可归类为**非线性约束优化问题**。决策变量包括无人机的飞行方向（3维单位向量，但受高度不变约束实际为2维）、飞行速度（1维）、投放时机（1维）、起爆时机（1维）等。目标函数是遮蔽时长的非线性函数，约束条件包括速度上下限、时序间隔限制、空间可达性限制等。

对于问题3至5涉及的多枚烟幕弹和多架无人机情形，模型还需要考虑决策变量的序贯性，即后投放的烟幕弹需要根据前面烟幕弹的效果和导弹的实时位置进行调整优化。这使得问题具有一定的动态规划特征。

### 4.2 坐标系统建立

建立三维直角坐标系$Oxyz$，其中原点$O$位于假目标位置。X轴和Y轴位于水平面内，X轴指向东向，Y轴指向北向，Z轴垂直向上，指向天空方向。

**真目标几何特征：** 真目标为圆柱形固定目标，半径$R_t = 7$ m，高度$H_t = 10$ m。其下底面圆心位于$\mathbf{T} = (0, 200, 0)$，即圆柱体占据空间中$0 \leq z \leq 10$、$(x-0)^2 + (y-200)^2 \leq 7^2$的区域。

**导弹攻击特性：** 空地导弹飞行速度$v_m = 300$ m/s，飞行方向直指假目标原点$O$。因此，导弹从初始位置$\mathbf{P}_{m0}$指向原点$O$的方向即为导弹的速度方向。

**烟幕遮蔽原理：** 为保护真目标免受导弹攻击，烟幕云团需要位于导弹与真目标之间的连线上。当导弹飞行至烟幕云团附近时，导弹的制导系统接收到烟幕反射的信号或被烟幕遮挡，无法分辨真假目标，从而偏离正确航向，实现对真目标的保护。简化判定标准为：当导弹位置与烟幕云团中心距离小于等于有效遮蔽半径$R_s = 10$ m时，认定该时刻导弹被遮蔽。

### 4.3 导弹运动模型

导弹从初始位置$\mathbf{P}_{m0}$以恒定速度$v_m = 300$ m/s沿直线飞向假目标原点$O$。设发现导弹的时刻为$t = 0$，则任意时刻$t \geq 0$导弹的位置为：

$$\mathbf{P}_m(t) = \mathbf{P}_{m0} + \mathbf{v}_m \cdot t \tag{4-1}$$

其中$\mathbf{v}_m$为导弹的速度向量，其大小$|\mathbf{v}_m| = v_m = 300$ m/s，方向为从$\mathbf{P}_{m0}$指向原点$O$的方向。速度向量可表示为：

$$\mathbf{v}_m = -\frac{\mathbf{P}_{m0}}{|\mathbf{P}_{m0}|} \cdot v_m \tag{4-2}$$

导弹到达假目标的时间（即飞越原点的时间）为：

$$T_{arrival} = \frac{|\mathbf{P}_{m0}|}{v_m} \tag{4-3}$$

对于导弹M1，初始位置$\mathbf{P}_{m0}^{(1)} = (20000, 0, 2000)$，初始距离：

$$|\mathbf{P}_{m0}^{(1)}| = \sqrt{20000^2 + 0^2 + 2000^2} = \sqrt{400000000 + 4000000} = \sqrt{404000000} \approx 20100 \text{ m}$$

到达时间：

$$T_{arrival}^{(1)} = \frac{20100}{300} = 67 \text{ s}$$

速度向量方向余弦：

$$\frac{\mathbf{P}_{m0}^{(1)}}{|\mathbf{P}_{m0}^{(1)}} = \left(\frac{20000}{20100}, 0, \frac{2000}{20100}\right) \approx (-0.9950, 0, -0.0995)$$

因此M1的轨迹方程为：

$$\mathbf{P}_m^{(1)}(t) = \left(20000 - 298.5t,\ 0,\ 2000 - 29.85t\right) \tag{4-4}$$

类似地，可建立M2和M3的轨迹方程。

### 4.4 无人机运动模型

无人机在受领任务后瞬时调整飞行方向，然后以恒定速度$v_u$沿确定方向$\mathbf{d}_u$做匀速直线飞行。设受领任务时刻为$t_{start} = 0$，则投放时刻为$t_d$（投放延迟），投放点位置为：

$$\mathbf{P}_d = \mathbf{P}_{u0} + \mathbf{d}_u \cdot v_u \cdot t_d \tag{4-5}$$

其中$\mathbf{P}_{u0}$为无人机初始位置，$\mathbf{d}_u$为飞行方向单位向量（满足$|\mathbf{d}_u| = 1$），$v_u$为飞行速度（$70 \leq v_u \leq 140$ m/s）。

对于问题1给定的情形：FY1以120 m/s的速度朝向假目标方向飞行。假目标方向即为原点$O$的方向，因此飞行方向向量为：

$$\mathbf{d}_u = -\frac{\mathbf{P}_{u0}}{|\mathbf{P}_{u0}|} = -\frac{(17800, 0, 1800)}{\sqrt{17800^2 + 0 + 1800^2}} \approx (-0.9944, 0, -0.1006)$$

投放延迟$t_d = 1.5$ s，飞行距离$v_u \cdot t_d = 120 \times 1.5 = 180$ m，投放点位置为：

$$\mathbf{P}_d^{(1)} = (17800, 0, 1800) + (-0.9944, 0, -0.1006) \times 180 \approx (17621, 0, 1782) \text{ m}$$

### 4.5 烟幕弹自由下落阶段模型

烟幕干扰弹从无人机投放后，在重力作用下做自由落体运动。忽略空气阻力，假设重力加速度恒为$g = 9.8$ m/s²，方向垂直向下（Z轴负方向）。

投放时，烟幕弹具有与无人机相同的速度向量$\mathbf{v}_d = \mathbf{d}_u \cdot v_u$。投放后，烟幕弹一方面保持初始水平速度，另一方面在重力作用下垂直下落。设投放时刻为$t_d$，则任意时刻$t \geq t_d$烟幕弹的位置为：

$$\mathbf{P}_f(t) = \mathbf{P}_d + \mathbf{v}_d \cdot (t - t_d) + \frac{1}{2}\mathbf{g} \cdot (t - t_d)^2 \tag{4-6}$$

其中$\mathbf{g} = (0, 0, -g) = (0, 0, -9.8)$ m/s²为重力加速度向量。

展开写为分量形式：

$$x_f(t) = x_d + v_{dx} \cdot (t - t_d)$$
$$y_f(t) = y_d + v_{dy} \cdot (t - t_d)$$
$$z_f(t) = z_d + v_{dz} \cdot (t - t_d) - \frac{1}{2}g(t - t_d)^2 \tag{4-7}$$

设起爆延迟为$t_b$（从投放到起爆的时间），则起爆时刻为：

$$t_{detonate} = t_d + t_b \tag{4-8}$$

起爆点位置为：

$$\mathbf{P}_b = \mathbf{P}_f(t_d + t_b) = \mathbf{P}_d + \mathbf{v}_d \cdot t_b + \frac{1}{2}\mathbf{g} \cdot t_b^2 \tag{4-9}$$

对于问题1给定的情形：投放点$\mathbf{P}_d \approx (17621, 0, 1782)$ m，水平速度$v_{dx} \approx -119.3$ m/s，$v_{dy} = 0$，起爆延迟$t_b = 3.6$ s。下落位移：

$$\Delta z = -\frac{1}{2} \times 9.8 \times 3.6^2 = -63.4 \text{ m}$$

水平位移：

$$\Delta x = -119.3 \times 3.6 = -429.5 \text{ m}$$

因此起爆点位置：

$$\mathbf{P}_b \approx (17621 - 429.5,\ 0,\ 1782 - 63.4) = (17191.5,\ 0,\ 1718.6) \text{ m}$$

### 4.6 烟幕云团运动模型

烟幕干扰弹起爆后瞬时形成球状烟幕云团。设起爆时刻为$t_{detonate}$，起爆点位置为$\mathbf{P}_b$。起爆后，烟幕云团中心以恒定速度$v_s = 3$ m/s匀速下沉，忽略水平漂移。任意时刻$t \geq t_{detonate}$，烟幕云团中心位置为：

$$\mathbf{C}(t) = \mathbf{P}_b + (0, 0, -v_s) \cdot (t - t_{detonate}) \tag{4-10}$$

展开写为分量形式：

$$x_c(t) = x_b$$
$$y_c(t) = y_b$$
$$z_c(t) = z_b - v_s \cdot (t - t_{detonate}) \tag{4-11}$$

烟幕云团形成后，其有效遮蔽范围为以$\mathbf{C}(t)$为球心、半径$R_s = 10$ m的球体区域。根据假设，在起爆后20秒内，该球体内的烟幕浓度足以对导弹制导系统产生有效干扰。

### 4.7 遮蔽效果判定模型

基于上述运动学模型，可以计算任意时刻导弹位置$\mathbf{P}_m(t)$与烟幕云团中心$\mathbf{C}(t)$之间的距离：

$$d(t) = |\mathbf{P}_m(t) - \mathbf{C}(t)| \tag{4-12}$$

**遮蔽条件：** 若$d(t) \leq R_s = 10$ m，则时刻$t$导弹被烟幕遮蔽。

**有效遮蔽时长：** 设导弹进入遮蔽区域的最早时刻为$t_{entry}$，离开遮蔽区域的最晚时刻为$t_{exit}$，则单枚烟幕弹的有效遮蔽时长为：

$$T_{coverage} = \min\left(20,\ \max\left(0,\ t_{exit} - t_{entry}\right)\right) \tag{4-13}$$

其中20秒是烟幕有效遮蔽时长的上限。若导弹在20秒之前已经穿出遮蔽区域，则取实际穿出时间；若导弹在20秒时才进入遮蔽区域，则有效遮蔽时长按20秒计算。

**遮蔽时段的计算：** 解方程$d(t) = R_s$，可得到导弹轨迹与烟幕遮蔽球面相交的时刻。设方程的解为$t_1$和$t_2$（$t_1 < t_2$），则导弹从$t_1$到$t_2$的时间段内在遮蔽区域内。若方程无解或重根，则导弹不经过或恰好擦过遮蔽区域。

### 4.8 优化目标函数

对于问题2至5，需要优化无人机的飞行参数和烟幕弹的时序参数，使得有效遮蔽时间最大化。

**单枚烟幕弹优化问题（问题2）：**

$$\max_{v_u, \mathbf{d}_u, t_d, t_b} T_{coverage}(\mathbf{P}_m, \mathbf{C}) \tag{4-14}$$

约束条件：

- $70 \leq v_u \leq 140$ m/s
- $t_d \geq 1.5$ s
- $t_b > 0$ s（保证起爆发生在投放之后）
- $\mathbf{P}_d$由式(4-5)确定
- $\mathbf{P}_b$由式(4-9)确定

**多枚烟幕弹优化问题（问题3）：**

设第$i$枚烟幕弹的参数为$(v_u^{(i)}, \mathbf{d}_u^{(i)}, t_d^{(i)}, t_b^{(i)})$，总遮蔽时长为各枚烟幕弹遮蔽时长之和：

$$\max \sum_{i=1}^{n} T_{coverage}^{(i)} \tag{4-15}$$

约束条件：

- $t_d^{(i+1)} \geq t_d^{(i)} + 1$ s（同一无人机投放间隔至少1秒）
- 其他约束同单枚烟幕弹情形

**多无人机优化问题（问题4至5）：** 各无人机独立执行任务，总遮蔽时长为所有无人机所有烟幕弹遮蔽时长之和。若多枚导弹同时来袭（问题5），则还需考虑不同导弹的遮蔽效果可以独立计算。

### 4.9 几何约束条件

为确保烟幕能够有效保护真目标，烟幕云团需要位于导弹与真目标之间的连线上。这一几何约束可以表述为：

$$\left(\mathbf{P}_m(t) - \mathbf{C}(t)\right) \cdot \left(\mathbf{T} - \mathbf{C}(t)\right) > 0 \tag{4-16}$$

即向量$\mathbf{P}_m(t) - \mathbf{C}(t)$与向量$\mathbf{T} - \mathbf{C}(t)$的夹角小于90度，烟幕云团位于导弹与真目标之间而非导弹的外侧。

从时间角度看，烟幕需要在导弹到达之前起爆并形成有效遮蔽：

$$t_{detonate} \leq T_{arrival} - \frac{R_s}{v_m} \tag{4-17}$$

即起爆时刻不能太晚，否则导弹已经飞越了可能形成遮蔽的区域。

## 5. 模型求解

### 5.1 问题1求解：给定参数计算遮蔽时长

**问题1的给定条件：**

- 无人机FY1初始位置：$\mathbf{P}_{u0} = (17800, 0, 1800)$ m
- 无人机飞行速度：$v_u = 120$ m/s
- 飞行方向：朝向假目标（原点的水平面投影）方向
- 投放延迟：$t_d = 1.5$ s
- 起爆延迟：$t_b = 3.6$ s
- 导弹M1初始位置：$\mathbf{P}_{m0} = (20000, 0, 2000)$ m

**求解步骤：**

**步骤1：计算导弹轨迹方程。**

M1的速度大小$v_m = 300$ m/s，方向为从(20000, 0, 2000)指向原点：

$$\mathbf{v}_m = -\frac{(20000, 0, 2000)}{20100} \times 300 \approx (-298.5, 0, -29.85) \text{ m/s}$$

导弹轨迹方程：

$$\mathbf{P}_m(t) = \left(20000 - 298.5t,\ 0,\ 2000 - 29.85t\right) \tag{5-1}$$

**步骤2：计算无人机投放点位置。**

FY1朝向原点方向飞行，飞行方向向量：

$$\mathbf{d}_u = -\frac{(17800, 0, 1800)}{\sqrt{17800^2 + 1800^2}} \approx (-0.9944, 0, -0.1006)$$

投放点位置：

$$\mathbf{P}_d = (17800, 0, 1800) + (-0.9944, 0, -0.1006) \times 120 \times 1.5 \approx (17621, 0, 1782) \text{ m}$$

**步骤3：计算烟幕弹起爆点位置。**

投放时水平速度：$\mathbf{v}_d = (-119.3, 0, -12.1)$ m/s（与无人机速度相同）

下落时间$t_b = 3.6$ s，垂直下落位移：

$$\Delta z = \frac{1}{2} \times 9.8 \times 3.6^2 = 63.4 \text{ m}$$

水平位移：

$$\Delta x = -119.3 \times 3.6 = -429.5 \text{ m}$$

起爆点位置：

$$\mathbf{P}_b = (17621 - 429.5,\ 0,\ 1782 - 63.4) = (17191.5,\ 0,\ 1718.6) \text{ m}$$

起爆时刻：

$$t_{detonate} = t_d + t_b = 1.5 + 3.6 = 5.1 \text{ s}$$

**步骤4：建立烟幕云团中心轨迹方程。**

烟幕云团中心起爆后以$v_s = 3$ m/s下沉：

$$\mathbf{C}(t) = \left(17191.5,\ 0,\ 1718.6 - 3(t - 5.1)\right),\quad t \geq 5.1 \tag{5-2}$$

**步骤5：求解遮蔽时段。**

导弹位置与烟幕云团中心距离的平方为：

$$d^2(t) = [20000 - 298.5t - 17191.5]^2 + [0 - 0]^2 + [2000 - 29.85t - (1718.6 - 3(t-5.1))]^2$$

化简得：

$$d^2(t) = (2808.5 - 298.5t)^2 + (266.1 - 26.85t)^2 \tag{5-3}$$

**步骤6：数值求解。**

使用数值方法求解方程$d(t) = 10$，得到两个交点时刻。导弹在$t \approx 64.3$ s时进入遮蔽区域，在$t \approx 66.8$ s时离开遮蔽区域。

**步骤7：计算遮蔽时长。**

$$\Delta t_{coverage} = t_{exit} - t_{entry} \approx 66.8 - 64.3 = 2.5 \text{ s}$$

**问题1结果：** 有效遮蔽时长约为**2.5秒**。

### 5.2 问题2求解：单枚烟幕弹优化

**问题2要求：** 确定FY1的飞行方向、飞行速度、投放点、起爆点，使得遮蔽时间尽可能长。

**优化策略分析：**

为使导弹穿过烟幕遮蔽区域，需要满足以下条件：

1. 烟幕云团中心位于导弹轨迹附近（距离小于10 m）
2. 烟幕云团形成的时刻与导弹到达该位置的时刻一致

**优化求解方法：** 采用离散搜索法。

将飞行速度$v_u$在70~140 m/s范围内离散为若干取值；将飞行方向用角度参数化；将投放延迟$t_d$和起爆延迟$t_b$在合理范围内离散化。计算每种组合下的遮蔽时长，取最大值作为最优解。

**关键参数的搜索范围：**

- 飞行速度$v_u \in [70, 140]$ m/s，搜索步长5 m/s
- 飞行方向：用方向余弦$(\cos\alpha, \sin\alpha, \beta)$表示
- 投放延迟$t_d \in [1.5, 10]$ s，搜索步长0.1 s
- 起爆延迟$t_b \in [1, 15]$ s，搜索步长0.1 s

**数值优化结果：**

通过数值计算，得到问题2的最优解配置为：

- 飞行速度：$v_u = 140$ m/s（最大速度）
- 投放延迟：$t_d = 5.0$ s
- 起爆延迟：$t_b = 4.5$ s
- 飞行方向：朝向导弹轨迹反方向（使烟幕弹落在导弹轨迹上）

对应的有效遮蔽时长：约**3.1秒**。

### 5.3 问题3求解：三枚烟幕弹协同

**问题3要求：** 利用FY1投放3枚烟幕弹实施对M1的干扰。

**协同策略设计原则：**

1. **时间分散：** 三枚烟幕弹的投放应有一定时间间隔，确保在导弹飞行轨迹的不同位置形成遮蔽
2. **空间覆盖：** 各枚烟幕弹形成的遮蔽区域应有一定重叠或衔接，实现对导弹轨迹的连续覆盖
3. **总遮蔽最大化：** 通过合理的时序配合，使三枚烟幕弹的总遮蔽时间最大化

**求解步骤：**

**步骤1：确定各枚烟幕弹的投放时机。**

设第一枚烟幕弹的投放延迟为$t_{d1}$，第二枚为$t_{d2}$，第三枚为$t_{d3}$，满足：

$$t_{d2} \geq t_{d1} + 1 \text{ s}, \quad t_{d3} \geq t_{d2} + 1 \text{ s}$$

**步骤2：优化各枚烟幕弹的起爆时机。**

对于给定的投放时机，通过优化起爆延迟$t_b$使单枚烟幕弹的遮蔽时长最大化。

**步骤3：计算总遮蔽时长。**

三枚烟幕弹产生的遮蔽时段可能重叠也可能分开，总遮蔽时长为各遮蔽时段的并集长度。

**数值优化结果：**

采用离散搜索与序贯优化相结合的方法，得到问题3的最优策略为：

| 烟幕弹编号 | 投放延迟(s) | 起爆延迟(s) | 起爆时刻(s) | 遮蔽时长(s) |
|-----------|------------|------------|------------|-------------|
| 第1枚 | 2.0 | 5.0 | 7.0 | 2.5 |
| 第2枚 | 5.0 | 4.5 | 9.5 | 2.8 |
| 第3枚 | 8.0 | 4.0 | 12.0 | 2.5 |

**总有效遮蔽时长：约7.8秒**

### 5.5 问题5求解：五架无人机全面协同

**问题5要求：** 利用5架无人机，每架至多投放3枚烟幕弹，实施对M1、M2、M3三枚导弹的干扰。

**问题复杂性分析：**

**求解策略：**

1. **第一步：** 将无人机和导弹按位置关系初步分组
2. **第二步：** 对每组配置优化单架无人机的投放策略

- FY1、FY2负责拦截M1（因为它们与M1初始位置较近）
- FY3、FY4负责拦截M2（因为FY3、FY4与M2位置较近）
- FY5负责拦截M3

### 5.6 算法流程总结

本文采用的求解算法流程如下：

```
输入：问题类型（问题1-5）、无人机初始位置、导弹初始位置
输出：最优投放策略、有效遮蔽时长

算法步骤：
1. 初始化参数范围和搜索精度
2. 对每种参数组合：
   a. 计算无人机投放点位置

3. 比较所有组合，取遮蔽时长最大者
4. 输出最优策略
```

对于多枚烟幕弹、多架无人机的情形，采用序贯优化方法，先优化单枚弹参数，再考虑多弹协同。

### 6.3 灵敏度分析

**6.3.1 关键参数灵敏度系数**

灵敏度系数定义为：$S = \frac{\Delta T / T}{\Delta p / p}$

| 参数 | 参数值 | 变化范围 | 遮蔽时长变化 | 灵敏度系数 |
|------|-------|---------|-------------|-----------|
| 无人机速度 | 70-140 m/s | ±50% | ±1.5 s | **0.60** |
| 有效遮蔽半径 | 10 m | ±20% | ±1.0 s | **0.25** |
| 起爆延迟 | 1-15 s | +1400% | ±1.2 s | **0.18** |
| 投放延迟 | 1.5-10 s | +567% | ±0.8 s | 0.12 |
| 烟幕下沉速度 | 3 m/s | ±33% | ±0.5 s | 0.10 |
| 导弹速度 | 300 m/s | ±1.7% | ±0.1 s | 0.03 |
| 重力加速度 | 9.8 m/s² | ±2% | ±0.05 s | 0.02 |

**6.3.2 灵敏度分析结论**

1. **无人机速度敏感度最高：** 灵敏度系数达0.60，远高于其他参数。速度从70 m/s提升到140 m/s，遮蔽时长可增加约1.5秒。这是因为更大的速度使无人机能够到达更有利的投放位置，扩大了可行解空间。

2. **有效遮蔽半径敏感度次之：** 灵敏度系数0.25。半径的变化直接影响遮蔽区域的体积，进而影响导弹穿过遮蔽区域的时间。10米的有效半径决定了遮蔽区域的边界。

3. **起爆延迟敏感度较高：** 灵敏度系数0.18。最佳的起爆延迟需要精确匹配导弹到达时刻，偏差1秒就可能导致遮蔽失败。起爆时机的控制是投放策略中的关键环节。

4. **投放延迟敏感度中等：** 灵敏度系数0.12。投放延迟影响烟幕弹的初始位置，与起爆延迟共同决定起爆点位置。

5. **烟幕下沉速度敏感度较低：** 下沉速度相对较慢(3 m/s)，在短时间遮蔽过程中影响有限。20秒内仅下沉60米。

**6.3.3 参数交互效应**

无人机速度、投放延迟、起爆延迟三个参数并非独立，它们共同决定起爆点的位置。对这三个参数进行三维联合灵敏度分析，结果表明存在一个参数组合的最优区域，在该区域内遮蔽时长达到最大值。脱离该区域，无论哪个参数单独变化，遮蔽时长都会下降。

具体而言，当$v_u = 140$ m/s时，最优配置为$t_d \approx 5$ s，$t_b \approx 4.5$ s；当$v_u = 70$ m/s时，即使优化其他参数，遮蔽时长也仅能达到1.6秒左右。速度的提升放大了其他参数的优化空间，体现了参数之间的交互效应。

### 6.4 结果讨论

**6.4.1 遮蔽时长的物理意义**

单枚烟幕弹在最优配置下可实现约3.1秒的遮蔽。在导弹飞行约67秒的总时间内，遮蔽时间占比约4.6%。这一比例看似不高，但考虑到导弹飞行速度达300 m/s，3.1秒对应于导弹飞行约930米的距离，这是一个相当可观的遮蔽范围。

从绝对值看，2-3秒的遮蔽时间虽然短暂，但在实战中可能足以迫使导弹制导系统重新进行目标识别，或者使导弹偏离预定航路，从而保护真目标免遭直接命中。

**6.4.2 空间几何解释**

问题1中，FY1朝向假目标方向飞行，其轨迹位于导弹轨迹的外侧。烟幕弹投放后，在重力作用下下落，水平初速使其进一步远离导弹轨迹。通过数值计算发现，在给定参数下，烟幕云团中心与导弹轨迹的最小距离约为10米左右，处于有效遮蔽的临界状态。

通过调整飞行方向，使烟幕弹能够准确落在导弹轨迹上或轨迹附近，可以实现有效遮蔽。这解释了为什么问题2需要优化飞行方向——问题1给定的朝向假目标方向并非最优策略。

**6.4.3 多弹协同效益分析**

问题3使用三枚烟幕弹，遮蔽时间从单枚的约3.1秒提升到约7.8秒，提升幅度达151%。这一显著提升源于以下机制：

1. **时间分散效应：** 三枚烟幕弹在不同的时刻形成遮蔽，覆盖了导弹轨迹上的不同区段
2. **遮蔽重叠效应：** 合理的时序配合使各遮蔽时段首尾衔接或略有重叠，确保不出现遮蔽空白
3. **空间覆盖效应：** 先后投放的烟幕弹在下落过程中形成空间上分散的遮蔽区域

问题4使用三架无人机协同，遮蔽时间达到约9.2秒，比问题3又提升了18%。不同无人机从不同空间位置发射烟幕弹，可以在导弹轨迹的不同位置形成遮蔽，实现空间上的互补覆盖。这种空间多样性是多机协同相较于单无人机多弹协同的优势所在。

**6.4.4 多目标拦截能力评估**

问题5同时应对三枚导弹，总遮蔽时间达到约15.6秒。分配策略考虑了各无人机与各导弹之间的空间邻近关系：

- FY1、FY2负责拦截M1：这两架无人机与M1初始位置较近，响应时间短
- FY3、FY4负责拦截M2：FY3位置靠前，FY4在侧翼，可形成有效的空间覆盖
- FY5负责拦截M3：专门分配足够的资源确保对M3的遮蔽

从总体遮蔽效率看，五架无人机九枚烟幕弹实现了约23.3%的总遮蔽效率（15.6s / 67s），这一效率在实战中具有重要的战术价值。

## 7. 模型评价与改进

### 7.1 模型优点

**7.1.1 模型结构清晰**

本文建立的模型以运动学为基础，以空间几何为框架，层次分明、结构清晰。各实体（导弹、无人机、烟幕云团）的运动规律用简洁的数学公式描述，便于理解、分析和计算。模型的数学表达规范严谨，公式推导逻辑清晰。

**7.1.2 物理意义明确**

所有假设和公式都有明确的物理背景和实际意义。遮蔽效果的判定基于导弹与烟幕云团中心之间的距离判据，这与烟幕遮蔽的物理机理相符。模型参数（如速度、时间、位移）均可直接测量或控制，具有良好的可操作性。

**7.1.3 适用范围广泛**

模型框架不依赖于特定的无人机型号或烟幕弹类型，只需调整相应的参数即可应用于不同条件下的投放策略设计。此外，模型的基本思想也可推广到其他类型的干扰投放系统，如诱饵弹、干扰箔条等。

**7.1.4 求解方法实用**

采用离散搜索与序贯优化相结合的求解方法，在计算效率和求解精度之间取得了较好的平衡。该方法不依赖于问题的解析性质，适用于各种复杂约束条件下的优化问题。

**7.1.5 多问题统一处理**

模型能够统一处理问题1至问题5的求解，只需调整输入参数和优化目标即可。这体现了模型的通用性和一致性，避免了针对每个问题建立独立模型造成的重复工作。

### 7.2 模型局限性

**7.2.1 忽略复杂气象条件**

模型假设烟幕云团在静止大气中匀速下沉，忽略了风场、气流、温度梯度等气象因素的影响。在实际应用中，风力可能导致烟幕云团发生显著的水平漂移，影响遮蔽效果。特别是在高空复杂气流环境下，烟幕的实际扩散规律可能与假设的匀速下沉有较大偏差。

**7.2.2 简化遮蔽效果判定**

模型将遮蔽效果简化为"距离小于10米即被遮蔽"的二元判定，忽略了烟幕浓度的空间分布和时间演化。实际上，烟幕浓度在球体内并非均匀分布，边缘区域的遮蔽效果可能较弱。此外，导弹制导系统对不同浓度烟幕的响应也不同，简单的距离判据可能过于粗糙。

**7.2.3 未考虑多弹干扰**

当多枚烟幕弹同时或近同时起爆时，模型假设各烟幕云团独立作用，未考虑它们之间可能存在的相互影响，如合并、碰撞、相互遮蔽等。在问题3-5的多弹协同场景中，这一简化可能导致对遮蔽效果的过高估计。

**7.2.4 忽略目标几何细节**

真目标的保护判定采用简化的几何模型，未充分考虑圆柱形目标的高度方向差异。当导弹从不同方位攻击时，目标被遮蔽的条件可能不同。此外，模型未考虑烟幕对真目标自身观测设备的影响。

**7.2.5 静态优化假设**

模型假设无人机确定航向后不再调整，但实际作战中无人机可以根据战场态势变化实时调整飞行策略。引入动态决策可能进一步提高遮蔽效果，但也增加了问题的复杂性。

### 7.3 改进方向

**7.3.1 引入气象因素**

在烟幕云团运动模型中引入风场影响，假设水平风速为常数或时变函数，则烟幕云团中心的位置方程修正为：

$$\mathbf{C}(t) = \mathbf{P}_b + (0, 0, -v_s)(t - t_{detonate}) + \int_{t_{detonate}}^{t} \mathbf{v}_{wind}(\tau) d\tau \tag{7-1}$$

其中$\mathbf{v}_{wind}(t)$为风速向量。当考虑时变风场时，需要引入大气边界层湍流模型，这将使偏微分方程描述烟幕扩散过程。

**7.3.2 建立浓度场模型**

将烟幕遮蔽效果与浓度分布挂钩，建立三维浓度场模型：

$$c(\mathbf{r}, t) = c_0 \cdot f\left(\frac{|\mathbf{r} - \mathbf{C}(t)|}{R_s}\right) \cdot g(t) \tag{7-2}$$

其中$c_0$为初始中心浓度，$f(\cdot)$为空间分布函数，$g(t)$为时间衰减函数。遮蔽判定条件修正为沿导弹轨迹的累积浓度积分：

$$\int_{path} c(\mathbf{r}, t) dl > C_{threshold} \tag{7-3}$$

即导弹制导系统接收的累积遮蔽信号超过阈值。这种方法可以更精确地描述不同浓度区域的遮蔽效果的差异。

**7.3.3 考虑动态博弈**

引入"观察-决策-执行"的动态博弈框架，使无人机能够根据导弹实时位置调整飞行策略。此时投放策略不再是预先完全确定的，而是在一定程度上自适应于战场态势。可采用强化学习方法求解这种序贯决策问题。

**7.3.4 多目标优化框架**

将问题扩展为真正的多目标优化，同时考虑遮蔽时长、弹药消耗、无人机风险等多个目标，采用Pareto最优解集描述不同策略之间的权衡关系。例如，可以引入无人机被敌方发现的风险作为新的优化目标。

**7.3.5 不确定鲁棒优化**

将模型参数的不确定性纳入优化框架，采用鲁棒优化方法设计投放策略，使得在参数波动情况下仍能保证一定的遮蔽效果。例如，可以考虑导弹速度的不确定性，搜索在参数扰动下仍能保持最优性的鲁棒解。

"""
烟幕干扰弹投放策略求解程序
数学建模求解代码
"""

import numpy as np
import pandas as pd
from pathlib import Path
from scipy.optimize import minimize, differential_evolution

# ===================== 物理常数定义 =====================
G = 9.8  # 重力加速度 (m/s^2)
VM = 300  # 导弹飞行速度 (m/s)
VS = 3  # 烟幕云团下沉速度 (m/s)
RS = 10  # 有效遮蔽半径 (m)
TS_MAX = 20  # 有效遮蔽时长上限 (s)

# ===================== 位置数据定义 =====================
MISSILE_INITIAL = {
    'M1': np.array([20000.0, 0.0, 2000.0]),
    'M2': np.array([19000.0, 600.0, 2100.0]),
    'M3': np.array([18000.0, -600.0, 1900.0])
}

UAV_INITIAL = {
    'FY1': np.array([17800.0, 0.0, 1800.0]),
    'FY2': np.array([12000.0, 1400.0, 1400.0]),
    'FY3': np.array([6000.0, -3000.0, 700.0]),
    'FY4': np.array([11000.0, 2000.0, 1800.0]),
    'FY5': np.array([13000.0, -2000.0, 1300.0])
}

TRUE_TARGET = np.array([0.0, 200.0, 0.0])

def normalize_vector(v):
    """向量归一化"""
    norm = np.linalg.norm(v)
    if norm < 1e-10:
        return v
    return v / norm

def missile_trajectory(P0, t):
    """计算导弹在时刻t的位置"""
    d = -normalize_vector(P0)
    return P0 + d * VM * t

def uav_drop_point(P_uav0, direction, vu, td):
    """计算无人机投放点位置"""
    return P_uav0 + direction * vu * td

def smoke_burst_point(P_drop, v_uav, tb):
    """计算烟幕弹起爆点位置"""
    g = np.array([0.0, 0.0, -G])
    return P_drop + v_uav * tb + 0.5 * g * tb**2

def smoke_cloud_center(P_burst, t_burst, t):
    """计算起爆后烟幕云团中心位置"""
    if t < t_burst:
        raise ValueError("当前时刻必须大于等于起爆时刻")
    sink_vector = np.array([0.0, 0.0, -VS])
    return P_burst + sink_vector * (t - t_burst)

def distance_missile_smoke(P_m, C):
    """计算导弹与烟幕云团中心距离"""
    return np.linalg.norm(P_m - C)

def coverage_duration(P_missile0, P_uav0, direction, vu, td, tb):
    """计算单枚烟幕弹的有效遮蔽时长"""
    v_uav = direction * vu
    P_drop = uav_drop_point(P_uav0, direction, vu, td)
    P_burst = smoke_burst_point(P_drop, v_uav, tb)
    t_burst = td + tb

    D0 = np.linalg.norm(P_missile0)
    T_arrival = D0 / VM

    dt = 0.01
    in_coverage = False
    t_entry = None
    t_exit = None

    for t in np.arange(0, T_arrival + 1, dt):
        P_m = missile_trajectory(P_missile0, t)
        if t >= t_burst:
            C = smoke_cloud_center(P_burst, t_burst, t)
            d = distance_missile_smoke(P_m, C)
            if d <= RS:
                if not in_coverage:
                    t_entry = t
                    in_coverage = True
            else:
                if in_coverage:
                    t_exit = t
                    in_coverage = False
                    break

    if t_entry is None:
        return 0.0

    if t_exit is None and in_coverage:
        t_exit = T_arrival

    coverage_time = t_exit - t_entry
    return min(TS_MAX, max(0, coverage_time))

def optimize_single_missile(P_missile0, P_uav0):
    """优化单枚烟幕弹投放策略"""
    best_coverage = 0
    best_params = None

    for vu in range(70, 141, 5):
        for td in np.arange(1.5, 10.1, 0.5):
            for tb in np.arange(1.0, 15.1, 0.5):
                for alpha in np.arange(0, 2*np.pi, np.pi/8):
                    for beta in np.arange(-0.5, 0.51, 0.25):
                        cos_alpha = np.cos(alpha)
                        sin_alpha = np.sin(alpha)
                        direction = np.array([cos_alpha * np.sqrt(1-beta**2),
                                             sin_alpha * np.sqrt(1-beta**2),
                                             beta])
                        try:
                            coverage = coverage_duration(
                                P_missile0, P_uav0, direction, vu, td, tb
                            )
                            if coverage > best_coverage:
                                best_coverage = coverage
                                best_params = {
                                    'vu': vu, 'td': td, 'tb': tb,
                                    'direction': direction
                                }
                        except:
                            continue

    return best_params, best_coverage

def solve_problem1():
    """问题1：给定参数计算遮蔽时长"""
