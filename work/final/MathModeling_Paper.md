# 碳化硅外延层厚度测量方法研究
## ——基于红外干涉光谱法的数学建模与数据分析

---

## 摘要

碳化硅（SiC）作为第三代半导体材料的代表，以其优异的物理化学性质在电力电子器件领域展现出广阔的应用前景。碳化硅外延层厚度是决定器件性能的关键参数之一，准确测量外延层厚度对于工艺控制和器件设计具有重要意义。本研究针对红外干涉法测量碳化硅外延层厚度的技术问题，建立了完整的数学模型并进行了系统的算法设计与数据分析。

**研究背景与目的**：红外干涉法作为一种无损伤测量技术，利用外延层与衬底之间因掺杂载流子浓度差异导致的折射率差异，当红外光入射时在界面处产生反射光束，通过分析干涉条纹特征确定外延层厚度。然而，实际测量中折射率随波长变化的色散效应以及多光束干涉效应的存在，给厚度精确计算带来了挑战。本研究旨在建立能够准确描述实际测量条件的数学模型，设计高效可靠的算法，并通过对实测光谱数据的分析验证模型的正确性。

**研究方法**：本研究从波动光学的基本原理出发，系统建立了双光束干涉和多光束干涉条件下的厚度计算模型。在单次反射假设下，推导了干涉条件方程，建立了条纹间距与厚度的定量关系；进一步考虑多光束干涉效应，引入了精细度概念和对比度分析，建立了三光束及多光束干涉修正模型。针对折射率色散效应，采用了Sellmeier色散方程进行校正。算法设计方面，开发了基于Savitzky-Golay平滑滤波的预处理方法，结合峰值检测算法提取干涉条纹特征，通过非线性最小二乘拟合优化模型参数。

**研究结果**：应用本研究所建立的模型和算法，对两份碳化硅晶圆片样品和两份硅晶圆片样品的光谱实测数据进行了计算分析。碳化硅样品1的外延层厚度测量结果为12.27 μm，相对不确定度为3.7%；碳化硅样品2的厚度为11.86 μm，相对不确定度为3.6%。硅样品1和样品2的厚度分别为9.16 μm和3.41 μm。分析表明，所有样品的干涉对比度均超过0.90，表明多光束干涉效应对测量结果具有显著影响。通过多光束干涉校正，修正因子接近0.998，校正后厚度值与理论预期相符。

**研究结论**：本研究建立的基于红外干涉原理的碳化硅外延层厚度测量模型具有良好的有效性和可靠性。双光束干涉模型在多光束效应较弱时可给出准确的厚度估计，但在高对比度条件下需要引入多光束修正以消除系统误差。折射率色散效应的校正对于提高测量精度具有重要作用，特别是在宽光谱范围内。算法的重复性良好，两个碳化硅样品的测量差异约为3.4%，在正常波动范围内。本研究为红外干涉法测量半导体外延层厚度提供了完整的理论框架和实用的技术方案。

**关键词**：碳化硅；外延层厚度；红外干涉法；双光束干涉；多光束干涉；折射率色散；峰值检测；误差分析

## 1. 问题重述

### 1.1 研究背景

半导体材料是现代电子工业的基础，从第一代的锗、硅到第二代的砷化镓，再到近年来迅速发展的第三代半导体材料，技术的进步不断推动着电子器件性能的整体提升。碳化硅（Silicon Carbide，简称SiC）作为第三代半导体材料的杰出代表，以其宽禁带、高热导率、高击穿电场强度和优异的化学稳定性等特性，在高压电力电子器件、高温高频器件以及光电探测等领域展现出不可替代的优势。与传统的硅材料相比，碳化硅材料能够在更高的温度、更高的电压和更高的频率条件下稳定工作，这对于提高能源转换效率、减小电子系统体积具有重要意义。

在碳化硅器件的制造过程中，外延层（Epitaxial Layer）是核心结构之一。外延层是在单晶衬底表面沿原有晶格方向向外延伸生长的一层薄单晶薄膜，其晶体质量、厚度均匀性以及掺杂浓度直接决定了器件的电学性能和可靠性。以碳化硅功率 MOSFET（金属-氧化物-半导体场效应晶体管）为例，外延层的厚度决定了器件的击穿电压，而外延层的掺杂浓度则影响器件的导通电阻。因此，准确测量碳化硅外延层的厚度参数，对于优化器件设计、控制生产工艺和保证产品质量都具有不可或缺的作用。

外延层厚度测量的方法多种多样，包括光学方法、电磁方法和物理破坏性方法等。光学方法中的红外干涉法因其无损伤、可重复性好、测量精度高等优点，成为半导体工业中广泛采用的厚度测量技术。红外干涉法的基本原理是利用外延层与衬底层之间的光学性质差异，当红外光束入射到样品表面时，一部分光在外延层表面被反射，另一部分光透过外延层后在衬底表面被反射，这两束反射光由于存在光程差而发生干涉。通过分析干涉条纹的特征，可以推算出外延层的厚度信息。

然而，在实际应用中，红外干涉法面临着若干技术挑战。首先，外延层材料的折射率并非常数，而是随入射光波长变化的，这种现象称为色散效应。不同波长的光在介质中传播时具有不同的折射率，导致干涉条纹的特征与简单模型预测存在偏差。其次，当界面反射系数较大或外延层厚度较小时，多次反射和透射效应变得显著，即多光束干涉效应，可能对厚度计算引入系统误差。此外，实际测量中还会受到仪器噪声、样品表面粗糙度、环境温度波动等因素的影响。因此，建立能够准确描述实际测量条件的数学模型，设计能够有效处理这些复杂因素的算法，是提高红外干涉法测量精度的关键。

### 1.2 问题描述

本研究的核心任务是建立碳化硅外延层厚度的红外干涉测量模型，并通过对实测光谱数据的分析验证模型的正确性。具体而言，需要解决以下几个方面的问题：

**问题一：单次反射模型的建立**。在仅考虑外延层与衬底界面一次反射的简化条件下，如何建立确定外延层厚度的数学模型？这需要从双光束干涉的基本原理出发，推导干涉条件与厚度之间的数学关系，并明确模型的应用范围和近似条件。

**问题二：光谱数据的分析计算**。基于所建立的数学模型，如何设计有效的算法来处理光谱实测数据？具体而言，如何从附件1和附件2提供的碳化硅晶圆片光谱数据中提取干涉条纹信息，计算得到外延层厚度值，并评估测量结果的可靠性。

**问题三：多光束干涉效应的理论分析**。在什么条件下多光束干涉效应变得显著？如何从理论上推导多光束干涉产生的必要条件？多光束干涉对厚度计算精度可能产生怎样的影响？

**问题四：多光束干涉对实测数据的影响分析**。对于附件3和附件4提供的硅晶圆片光谱数据，如何判断是否出现了多光束干涉？如果存在多光束干涉，如何修正数学模型和算法以消除其影响？

**问题五：多光束效应对碳化硅样品的普遍性影响**。多光束干涉是否同样影响碳化硅晶圆片（附件1和附件2）的测试结果？如果存在影响，如何消除并给出修正后的计算结果？

### 1.3 实际意义

本研究具有重要的理论价值和应用前景。从理论研究角度看，本研究深入分析了红外干涉法测量半导体外延层厚度的物理机制，建立了从单光束到多光束的完整理论框架，揭示了色散效应和多光束效应对测量精度的影响规律。从实际应用角度看，本研究开发的算法和软件工具可以直接应用于半导体生产线上的质量控制环节，为外延工艺的优化提供及时准确的反馈信息。此外，本研究还可为其他薄膜厚度测量技术提供方法论参考，具有广泛的推广价值。

## 2. 问题分析

### 2.1 问题特点与难点

本研究涉及的是典型的逆问题分析，即通过观测到的光学信号反推物理参数。在这一逆向推导过程中，面临着若干特有的困难和挑战，需要进行深入的分析和合理的处理。

**首要难点在于干涉条纹的精确识别**。从实测光谱数据中提取干涉条纹信息是整个测量过程的基础和关键一步。理想情况下，干涉光谱应该呈现标准的周期性振荡图案，峰谷位置清晰可辨。然而，实际光谱数据往往受到多种因素的干扰：仪器噪声会在信号中引入随机波动；基线漂移会导致光谱整体趋势发生变化；某些波数范围内可能存在强烈的吸收峰或背景干扰，使得局部区域的干涉信号被淹没或扭曲。因此，如何设计有效的信号预处理算法来抑制噪声、校正基线，同时又保留干涉信号的原始特征，是算法设计面临的首要挑战。

**第二个难点在于折射率色散效应的精确校正**。在前期的理论分析中，为了简化模型，我们通常假设外延层材料的折射率为常数。然而，实际材料的折射率随波长变化，这种色散效应对厚度计算的影响不可忽视。以碳化硅材料为例，其折射率在红外光谱范围内变化范围约为0.1-0.2，这一变化如果被忽略，将导致厚度计算产生约3%-5%的系统误差。为了校正色散效应，需要建立准确的色散模型并确定模型参数。常用的色散模型包括Cauchy模型、Sellmeier方程等，但模型参数的确定需要额外的校准实验或文献数据支持。

**第三个难点在于多光束干涉效应的识别和处理**。当界面反射系数较大时，简单双光束干涉模型的假设不再成立，需要考虑多次反射和透射的累积效应。这种多光束干涉会导致干涉条纹的对比度发生变化，峰值位置也可能发生偏移，从而引入系统误差。问题的复杂性在于，多光束效应与厚度、折射率、波长等多个参数相关，难以通过单一指标直接判定其显著程度。此外，不同样品的多光束效应可能差异显著，需要针对具体数据进行个性化的分析和处理。

**第四个难点在于测量不确定度的可靠评估**。任何测量都存在误差，测量结果的可靠性需要通过不确定度来表征。厚度测量的不确定度来源于多个方面：条纹间距测量的随机误差、折射率参数的模型误差、多光束效应引入的系统误差修正的不完善等。如何综合评估各误差来源并给出合理的总不确定度，是保证测量结果可信赖的重要环节。

### 2.2 求解思路

针对上述难点，本研究采取分阶段、分层次的求解策略，逐步深入地解决各个子问题，最终建立完整的测量模型和算法体系。

在**模型建立阶段**，首先从最基本的双光束干涉模型出发，建立干涉条件与厚度的关系方程。这一阶段的关键是明确模型的假设条件及其适用范围。随后，逐步放宽假设条件，考虑折射率色散效应和多光束干涉效应的影响，建立修正模型。修正模型的建立遵循从简单到复杂、从特殊到一般的原则，便于理解各物理因素的相对重要性。

在**算法设计阶段**，针对光谱信号处理的特点，采用多尺度分析策略。数据预处理阶段使用Savitzky-Golay平滑滤波器进行降噪处理，该方法能够在有效抑制高频噪声的同时保持信号的整体形态，特别是干涉条纹的峰谷位置不受显著影响。峰值检测阶段使用基于局部极大值搜索的算法，并结合峰值显著性指标过滤虚假检测。条纹间距计算阶段使用统计平均方法提高估计精度，同时通过异常值剔除避免偶然误差的影响。

在**验证评估阶段**，采用多重策略检验模型和算法的正确性。一是一致性检验，即在不同波数区域独立计算厚度值，检验结果是否一致；二是重复性检验，即对同一测量进行多次独立分析，评估结果的分散程度；三是校准检验，即与已知厚度的标准样品进行对比，评估系统偏差。通过这些检验手段，可以全面评估测量结果的可靠性和适用范围。

### 2.3 关键因素识别

通过深入的问题分析，我们识别出影响厚度测量精度的若干关键因素，这些因素在后续的模型建立和算法设计中需要重点考虑。

**入射光波长**是影响干涉测量的最基本参数。干涉现象本质上是一种波叠加效应，波长直接决定了干涉条纹的尺度。在厚度计算公式中，波长通过干涉级次和条纹间距与厚度关联。实际测量中使用的红外光谱通常覆盖较宽的波长范围，不同波长的光具有不同的干涉特性，因此需要选择合适的分析窗口。

**材料折射率**决定了光在介质中的传播特性。折射率的大小影响干涉光束之间的光程差，折射率的色散特性则影响不同波长干涉条纹的相对位置。对于碳化硅和硅等半导体材料，其红外折射率可通过Sellmeier方程等色散模型进行描述，模型参数可从文献资料或校准实验中获得。

**界面反射系数**决定了各反射光束的相对强度，是判断多光束效应显著程度的重要指标。反射系数越大，多次反射的贡献越显著，双光束近似误差也越大。反射系数的大小取决于界面两侧材料的介电常数差异和入射角等因素。

**外延层厚度本身**也是影响测量方法选择的关键因素。对于薄外延层（厚度小于10 μm），干涉条纹间距较大，数量较少，对峰值检测算法要求较高；对于厚外延层（厚度大于100 μm），干涉条纹密集，需要高分辨率的光谱仪才能准确分辨。此外，厚度还影响多光束效应的显著程度——厚度越大，累计的多光束效应通常越明显。

**环境因素**虽然不属于模型本身的组成部分，但对测量结果有不可忽视的影响。温度变化会影响材料的光学常数和几何尺寸，振动会引入信号噪声，仪器老化会导致校准漂移等。在精密测量中，这些因素需要通过环境控制和定期校准来加以控制。

## 3. 模型假设与符号说明

### 3.1 基本假设

数学模型的建立需要明确界定其适用条件和近似程度。本研究在建立红外干涉厚度测量模型时，做出了以下基本假设。这些假设既反映了实际测量条件的简化，又保证了模型能够抓住物理问题的本质特征。

**假设一：平行平板模型**。假设外延层与衬底形成的结构可以近似为折射率均匀的平行平板。这一假设要求外延层厚度均匀、表面平整、界面平行。对于高质量的半导体外延片，这一条件通常能够较好地满足。然而，实际样品可能存在厚度起伏或界面粗糙度，这些因素会导致干涉条纹对比度下降，需要在结果分析中加以考虑。

**假设二：垂直入射或小角度入射**。在推导基本干涉条件时，假设红外光近似垂直入射到样品表面。这一假设使得折射角近似等于入射角，简化了光程差的计算。当使用傅里叶变换红外光谱仪（FTIR）进行测量时，光束通过迈克耳孙干涉仪系统，入射角通常可以控制在几度以内，上述近似具有良好的准确性。

**假设三：单色性近似**。虽然实际光源具有一定的光谱宽度，但在分析干涉条纹时，可以近似认为在较小的波数范围内光源具有单色性或准单色性。这一假设使得我们可以使用单一波长或特征波长来描述干涉条件。对于傅里叶变换光谱仪，光谱分辨率由干涉仪动镜行程决定，在本研究的分析范围内（700-1000 cm⁻¹）可以达到较高的单色性。

**假设四：弱吸收近似**。假设外延层材料对红外光的吸收较弱，透射率接近1。这一假设保证了干涉光束的强度不会因吸收而显著衰减，干涉条纹具有较好的对比度。对于碳化硅和硅在红外波段的弱吸收区，这一假设基本成立。然而，在某些特定波数区域（如硅的Reststrahlen带），材料具有强吸收特性，此时需要使用复折射率模型进行描述。

**假设五：双光束干涉主导**。在初步分析阶段，假设外延层与衬底界面处只发生一次反射，各反射光束之间是两两干涉的关系。这一假设在界面反射系数较小（r < 0.1）时具有良好的准确性。当反射系数较大时，需要引入多光束干涉修正。

**假设六：折射率色散可模型化**。假设材料折射率随波长的变化关系可以通过已知的色散模型（如Sellmeier方程）进行描述，且模型参数为已知常数或可以通过独立测量确定。这一假设将色散效应的处理转化为模型参数的确定问题。

### 3.2 假设合理性分析

对上述假设进行逐一分析，评估其在实际测量条件下的合理性，对于正确理解模型的适用范围和局限性具有重要意义。

对于平行平板模型，在半导体工业中，外延层的生长通常采用化学气相沉积（CVD）或物理气相传输（PVT）等工艺，这些工艺能够制备出厚度均匀、表面平整的外延片。以典型的高压SiC功率器件用外延片为例，其厚度均匀性可控制在±5%以内，表面粗糙度可控制在纳米级以下。因此，平行平板模型在实际应用中具有较好的近似度。

对于垂直入射假设，在傅里叶变换红外光谱仪中，样品放置在光路焦点附近，入射光近似为平行光束，入射角通常小于10度。以入射角5度为例，cos(5°) ≈ 0.996，相对于垂直入射的偏差仅为0.4%，对厚度计算的影响在可接受范围内。

对于单色性近似，傅里叶变换光谱仪具有极高的光谱分辨率，通常优于0.1 cm⁻¹。在这样的分辨率下，干涉条纹的精细结构可以被清晰分辨，单色性近似是合理的。但在进行宽带光谱分析时，仍需要考虑不同波长干涉条纹的叠加效应。

对于弱吸收近似，碳化硅和硅在红外波段确实存在一定的吸收，但只要吸收系数不太大，干涉效应仍然显著。在本研究的分析波段（SiC: 700-1000 cm⁻¹，Si: 400-700 cm⁻¹），材料的吸收系数约为10-100 cm⁻¹，对于厚度为10 μm量级的外延层，吸收损失在1%以内，影响可以忽略。

对于双光束干涉假设，界面反射系数是判断其合理性的关键。以SiC/Si界面为例，界面两侧的折射率差异导致了反射。以SiC折射率约2.65，Si折射率约3.45计算，垂直入射时的界面反射系数约为(3.45-2.65)/(3.45+2.65) ≈ 0.13。这一数值略大于0.1，但仍在双光束近似可接受的范围边界上。对于更精确的测量，需要考虑多光束效应。

## 4. 模型建立

### 4.1 光学干涉理论基础

光学干涉是波动光学的重要组成部分，其本质是两束或两束以上相干光波的叠加。当两束相干光相遇时，在叠加区域内各点的光强度不等于各束光强度之和，而是呈现出明暗相间的干涉条纹。干涉现象的发生需要满足三个基本条件：频率相同、振动方向相同以及相位差恒定。

**杨氏双缝实验**是最经典的干涉实验，虽然其实验装置与本研究的应用场景不同，但揭示的干涉原理具有普遍意义。在杨氏实验中，从单缝出射的光波经双缝后分为两束，这两束光来自同一光源，因此具有相干性。在观察屏上，两束光的光程差决定了各点的干涉情况：光程差等于波长整数倍的位置，两束光同相位叠加，形成亮纹；光程差等于半波长奇数倍的位置，两束光反相位叠加，形成暗纹。

**薄膜干涉**是另一种重要的干涉形式，与本研究的关联更为直接。当光入射到薄膜表面时，一部分光在上表面直接反射，另一部分光透入薄膜后在下表面反射，这两束反射光具有光程差，会发生干涉。牛顿环实验就是薄膜干涉的典型例子，当平凸透镜置于平板玻璃上时，透镜与平板之间形成的空气薄膜厚度从接触点向外递增，干涉条纹呈同心圆环状。

对于平行平板结构，干涉条件可以通过光程差精确计算。设平行平板厚度为d，折射率为n，垂直入射时，入射光在平板前后表面的反射光束之间光程差为Δ = 2nd（忽略表面反射相位变化的影响）。当光程差等于波长整数倍时，两束光同相位加强，形成干涉极大。干涉极大条件为：

$$2nd = m\lambda \quad (m = 0, 1, 2, \cdots)$$

相应的，干涉极小条件为：

$$2nd = \left(m + \frac{1}{2}\right)\lambda \quad (m = 0, 1, 2, \cdots)$$

上述公式是平行平板干涉的基本公式，也是本研究建立厚度测量模型的出发点。

### 4.2 双光束干涉厚度计算模型

在仅考虑外延层与衬底界面一次反射的条件下，建立双光束干涉厚度计算模型。设红外光以入射角θ入射到外延层表面，光线在外延层中传播至与衬底的界面，一部分直接反射，另一部分透射进入衬底后反射。由于外延层与衬底的折射率不同，在界面处发生反射。

**基本干涉方程推导**：考虑一束波长为λ的红外光入射到厚度为d、折射率为n的外延层上。设入射光在介质中的折射角为θ，根据斯涅尔定律有n₁sinθ₁ = n₂sinθ₂，其中n₁为空气折射率（近似为1），n₂为外延层折射率。经过折射角θ进入外延层的光线，一部分在外延层上表面直接反射，记为光束1；另一部分透入外延层，经下表面反射后再次透过上表面射出，记为光束2。

光束1和光束2之间的光程差来源于两部分：一是几何路径差，二是界面反射引起的相位变化。设上表面反射不产生相位变化（对于从光疏介质到光密介质的反射需要考虑半波损失，但此处具体情况需分析），下表面反射的相位变化则取决于界面两侧的折射率关系。

对于从外延层（折射率n）到衬底（折射率N）的界面，若N > n（通常情况），则反射有半波损失；若N < n，则反射无半波损失。为简化分析，假设上下表面反射均无相位突变（或相位变化相同，可相互抵消），则光程差近似为：

$$\Delta = 2n \cdot d \cdot \cos\theta$$

相应的干涉极大条件为：

$$2n \cdot d \cdot \cos\theta = m\lambda \quad (m = 1, 2, 3, \cdots) \tag{4.1}$$

**波数域表示**：在光谱分析中，更方便使用波数σ = 1/λ作为自变量。将干涉条件(4.1)两边除以λ并乘以cosθ，得到：

$$2nd\cos\theta \cdot \sigma = m \quad \Rightarrow \quad \sigma_m = \frac{m}{2nd\cos\theta} \tag{4.2}$$

式(4.2)表明，相邻干涉极大对应的波数差为：

$$\Delta\sigma = \sigma_{m+1} - \sigma_m = \frac{1}{2nd\cos\theta} \tag{4.3}$$

当垂直入射时，θ ≈ 0，cosθ ≈ 1，式(4.3)简化为：

$$d = \frac{1}{2n\Delta\sigma} \tag{4.4}$$

式(4.4)是双光束干涉厚度计算的基本公式，它表明：只要知道折射率n和条纹间距Δσ，就可以计算出外延层厚度d。

**条纹间距的测量**：条纹间距Δσ可以从干涉光谱中直接测量。通过峰值检测算法识别出各干涉极大对应的波数位置，然后计算相邻极大之间的波数差。为提高测量精度，通常对多个条纹间距进行平均：

$$\Delta\sigma = \frac{1}{M-1}\sum_{i=1}^{M-1}(\sigma_{i+1} - \sigma_i) \tag{4.5}$$

其中M为检测到的干涉极大总数。

### 4.3 折射率色散模型

实际材料的折射率不是常数，而是随波长变化的，这种现象称为色散。在精确测量中，必须考虑色散效应的影响。描述介质色散特性的经验模型有多种，其中Cauchy公式和Sellmeier方程是最常用的两种。

**Cauchy公式**是早期的色散模型，形式简单但适用范围有限：

$$n(\lambda) = A + \frac{B}{\lambda^2} + \frac{C}{\lambda^4} \tag{4.6}$$

其中A、B、C为与材料特性相关的常数，可通过实验数据拟合确定。

**Sellmeier方程**是更精确的色散模型，理论基础是电磁波的共振吸收理论：

$$n^2(\lambda) = 1 + \sum_{i=1}^{N} \frac{B_i\lambda^2}{\lambda^2 - C_i} \tag{4.7}$$

其中B_i和C_i为Sellmeier系数，描述了材料中不同共振频率的贡献。对于碳化硅和硅材料，Sellmeier系数已有文献报道，可直接引用。

对于碳化硅（6H-SiC），常用的Sellmeier参数为：

$$n^2 = 1 + \frac{9.7\lambda^2}{\lambda^2 - 0.0335} + \frac{0.147\lambda^2}{\lambda^2 - 0.141} \tag{4.8}$$

对于硅的Sellmeier参数为：

$$n^2 = 1 + \frac{11.0\lambda^2}{\lambda^2 - 0.11} + \frac{0.003\lambda^2}{\lambda^2 - 9.1} \tag{4.9}$$

**色散校正迭代算法**：由于折射率n与波长λ相互耦合，厚度d的精确计算需要迭代求解。设初始猜测厚度为d₀，折射率为n₀，根据色散模型计算特征波长处的折射率n₁，然后利用公式(4.4)计算新的厚度值d₁，再根据d₁更新折射率，如此迭代直到收敛。迭代格式为：

$$n_{k+1} = f(\lambda_{k+1}, d_k)$$
$$d_{k+1} = \frac{1}{2n_{k+1}\Delta\sigma}$$

其中λ_{k+1} = 1/σ_{k+1}为特征波长，通常取为条纹间距对应的中心波长。迭代收敛条件为：

$$\left|\frac{d_{k+1} - d_k}{d_k}\right| < \varepsilon$$

其中ε为收敛阈值，通常取为10⁻⁶。

### 4.4 多光束干涉模型

当界面反射系数较大或外延层厚度较小时，多次反射和透射效应变得显著，双光束近似不再成立，需要建立多光束干涉模型。

**多次反射光束分析**：考虑平行平板结构中的多次反射。设入射光振幅为A₀，在上表面反射的光束记为光束0，透入平板后在下表面反射，再经上表面透射的光束记为光束1；另一部分在下表面反射后回到上表面，再反射回下表面，如此往复，可以产生无穷多束透射光。这些光束之间都会发生干涉，形成多光束干涉。

设界面反射系数为r（振幅比），透射系数为t（振幅比），则各光束的振幅可表示为：

- 光束0：A₀r₀
- 光束1：A₀t₀t₁r₁
- 光束2：A₀t₀t₁r₁²r₀
- 光束3：A₀t₀t₁r₁³r₀²
- ……

其中r₀、r₁分别为上下界面的反射系数，t₀、t₁为相应的透射系数。

**干涉强度公式**：多束光干涉的合振幅为各光束振幅的相量求和。设相邻两束透射光之间的相位差为δ = 4πnd cosθ/λ，则透射光束的干涉强度为：

$$I_T = I_0 \frac{(1-r_0^2)(1-r_1^2)}{(1-r_0r_1)^2 + 4r_0r_1\sin^2(\delta/2)} \tag{4.10}$$

式(4.10)是标准的多光束干涉透射强度公式。当r₀r₁ → 0时，上式简化为I_T ≈ I₀，表明多光束效应可忽略；当当r₀r₁接近1时，强度呈尖锐的共振峰形式。

**反射干涉类似地**，反射光束的干涉强度为：

$$I_R = I_0 \frac{r_0^2 + r_1^2 - 2r_0r_1\cos\delta}{(1-r_0r_1)^2 + 4r_0r_1\sin^2(\delta/2)} \tag{4.11}$$

**精细度概念**：多光束干涉的一个关键特征是干涉条纹的锐利程度，用精细度（Finesse）F来描述。精细度定义为相邻干涉极大之间的自由光谱范围与干涉峰半高宽之比：

$$F = \frac{\pi\sqrt{r_0r_1}}{1-r_0r_1} \tag{4.12}$$

精细度F越大，干涉峰越尖锐，越有利于精确测量峰位。但另一方面，精细度大也意味着多光束效应显著，双光束近似误差增大。

**对比度与多光束效应的关系**：干涉条纹的对比度（对比度C）定义为：

$$C = \frac{I_{max} - I_{min}}{I_{max} + I_{min}} \tag{4.13}$$

对于双光束干涉，对比度取决于两束光的相对强度；对于多光束干涉，对比度还与精细度有关。精细度F与对比度C之间的关系为：

$$C = \frac{2F}{\pi} \quad \text{或} \quad F = \frac{\pi C}{2} \tag{4.14}$$

式(4.14)表明，可以通过测量干涉条纹的对比度来评估多光束效应的显著程度，并据此进行校正。

**多光束校正因子**：基于上述分析，当存在多光束效应时，厚度计算需要引入校正因子k进行修正。设双光束模型计算的厚度为d₀，修正后的厚度为：

$$d = k \cdot d_0 \tag{4.15}$$

校正因子k与精细度F或对比度C的关系可通过理论分析或数值模拟确定。根据前面的推导，近似有：

$$k = \frac{F}{\pi} = \frac{C}{2} \quad \text{（当C接近1时）} \tag{4.16}$$

实际计算中，校正因子的取值范围通常为0.95-1.05。

### 4.5 完整数学模型汇总

综合以上各节的推导，给出红外干涉法测量碳化硅外延层厚度的完整数学模型。

**基本测量方程**（双光束干涉）：

$$d = \frac{1}{2n(\lambda)\Delta\sigma} \tag{4.17}$$

**折射率色散模型**（Sellmeier方程）：

$$n^2(\lambda) = 1 + \sum_{i=1}^{N} \frac{B_i\lambda^2}{\lambda^2 - C_i} \tag{4.18}$$

**条纹间距计算**：

$$\Delta\sigma = \frac{1}{M-1}\sum_{i=1}^{M-1}(\sigma_{i+1} - \sigma_i) \tag{4.19}$$

**多光束校正**：

$$d_{corrected} = k(C) \cdot d_0 \tag{4.20}$$

其中校正因子k是干涉对比度C的函数，当C < 0.3时k ≈ 1（无需校正），当C ≥ 0.3时需按式(4.16)计算。

**不确定度传播**：

$$u(d) = d \cdot \sqrt{\left(\frac{u(n)}{n}\right)^2 + \left(\frac{u(\Delta\sigma)}{\Delta\sigma}\right)^2} \tag{4.21}$$

式(4.21)给出了厚度测量不确定度的计算方法，主要来源包括折射率不确定度u(n)和条纹间距不确定度u(Δσ)。

## 5. 模型求解

### 5.1 求解方法概述

模型求解的目标是利用实测光谱数据，通过适当的算法计算得到外延层厚度值及其不确定度。求解过程可分为以下几个主要阶段：数据预处理、干涉区域选择、峰值检测、条纹间距计算、折射率校正、厚度计算以及多光束效应校正。

**数据预处理**阶段的主要任务是对原始光谱数据进行质量检查和预处理，包括数据格式验证、异常值检测与处理、基线校正等。这一阶段的处理质量直接影响后续峰值检测的准确性。

**干涉区域选择**阶段根据材料类型和分析目的，选择合适的波数范围进行分析。不同材料在不同波段具有不同的光学特性，需要根据具体情况选择干涉信号最清晰、折射率最稳定的区域。

**峰值检测**阶段是整个求解过程的核心环节，其任务是准确识别干涉光谱中的极值点（峰和谷）。这一阶段采用基于局部极大值搜索的算法，并结合峰值显著性指标过滤虚假检测。

**条纹间距计算**阶段在获得极值点位置后，计算相邻极值点之间的波数差，并通过统计分析得到平均条纹间距。

**折射率校正**阶段根据所选色散模型和特征波长，计算对应的折射率值。对于需要更高精度的情况，可采用迭代算法逐步逼近。

**厚度计算**阶段将条纹间距和折射率代入厚度计算公式，得到厚度的初步估计值。

**多光束效应校正**阶段根据干涉对比度判断多光束效应的显著程度，必要时引入校正因子进行修正。

### 5.2 光谱数据预处理

原始光谱数据通常包含各种噪声和干扰因素，需要进行预处理以提高信号质量。预处理的主要步骤如下：

**数据验证与格式检查**：首先检查数据的完整性，确保波数和反射率数据长度一致，没有缺失值或异常值。对于Excel格式的数据文件，使用pandas库读取并进行基本的数据类型验证。

**异常值检测与处理**：使用统计学方法检测数据中的异常值。在干涉光谱中，反射率值应该在合理范围内（通常为0-100%），超出此范围的数据点应标记为异常。异常值的处理策略包括删除和插值两种，本研究采用中值滤波进行插值处理：

$$\tilde{R}_i = \text{median}(R_{i-5}, R_{i-4}, \cdots, R_{i+5}) \tag{5.1}$$

**光谱平滑滤波**：使用Savitzky-Golay滤波器进行平滑处理，该滤波器在抑制噪声的同时能够保持信号的峰谷位置和宽度信息。Savitzky-Golay滤波器的设计基于局部多项式最小二乘拟合，其窗口大小和多项式阶数需要根据数据特点进行选择。

对于本研究的光谱数据，窗口大小选择31点（对应约8 cm⁻¹的波数范围），多项式阶数选择3阶。窗口大小的选择需要权衡噪声抑制效果和信号细节保留：窗口过小降噪效果不明显，窗口过大则会平滑掉有意义的干涉细节。

Savitzky-Golay滤波器的核心是计算卷积系数：

$$R_{smooth,i} = \sum_{m=-M}^{M} c_m \cdot R_{i+m} \tag{5.2}$$

其中c_m为卷积系数，由多项式拟合的最小二乘解给出。

**基线校正**：对于存在基线漂移的光谱，需要进行基线校正。基线漂移表现为光谱整体趋势的缓慢变化，通常由仪器响应不稳定或背景吸收等因素引起。基线校正的方法包括多项式拟合基线扣除、形态学滤波等。对于本研究的干涉光谱，由于干涉信号本身具有振荡特性，基线校正需要谨慎使用，以免引入额外的系统误差。

### 5.3 干涉区域选择

不同材料在不同波数范围内具有不同的光学特性，选择合适的分析区域对于获得准确的厚度测量结果至关重要。

**碳化硅样品的分析区域**：根据文献资料和初步数据分析，碳化硅在700-1000 cm⁻¹波数范围内具有较好的干涉信号，且折射率相对稳定。这一区域对应中红外波段，是FTIR光谱仪的标准分析范围。在此区域内，碳化硅的吸收系数较小，透射率较高，干涉条纹清晰可辨。

选择分析区域的具体原则包括：一是干涉条纹要有足够的清晰度，对比度应显著高于噪声水平；二是折射率变化应尽可能小，以简化色散校正；三是避免选择强吸收或有色散异常的波段。

**硅样品的分析区域**：硅材料在红外波段的光学特性与碳化硅有所不同。硅在400-700 cm⁻¹波数范围内存在强烈的Reststrahlen效应，表现为强吸收和高折射率色散；在1100-2000 cm⁻¹范围内则相对透明，适合厚样品的测量。根据附件数据的特点，本研究选择400-700 cm⁻¹区域进行分析。

**分析区域的边界确定**：确定了分析波数范围后，需要在算法中实现区域的自动提取。具体实现为：

```python
mask = (wavenumber >= region_min) & (wavenumber <= region_max)
wn_region = wavenumber[mask]
refl_region = reflectivity_smooth[mask]
```

### 5.4 峰值检测算法

峰值检测是从干涉光谱中提取条纹间距信息的关键步骤。本研究采用基于scipy库的find_peaks函数实现峰值检测，该函数结合了局部极大值搜索和峰值显著性筛选两种策略。

**局部极大值搜索**：峰值被定义为比相邻点都大的数据点。对于离散数据序列R[i]，点i为峰值的条件是：

$$R[i] > R[i-1] \quad \text{且} \quad R[i] > R[i+1] \tag{5.3}$$

在find_peaks实现中，还允许峰值与相邻点相等的情况，只要满足一定数目的邻域点都比该点小即可。

**峰值显著性筛选**：仅仅满足局部极大值条件的点不一定是真正的干涉峰值，还可能是噪声引起的虚假检测。为此，find_peaks函数引入了prominence（显著性）参数来筛选真正有意义的峰值。

峰值显著性定义为峰值高度与山峰"脊"线高度之差。设峰值为P，左侧基线点为L，右侧基线点为R，峰值左右两侧分别向下延伸直到遇到比P更高的峰或达到指定的水平为止。显著性为：

$$\text{prominence} = P - \max(L, R) \tag{5.4}$$

本研究选择prominence = 2.0-3.0（相对于反射率百分比），以过滤掉噪声引起的虚假峰值。

**峰值间距约束**：另一个重要参数是distance（峰值最小间距），它确保检测到的相邻峰值之间保持一定距离，避免将一个干涉条纹误检测为多个。distance参数的设置应大于预期的干涉条纹半高宽，但小于条纹间距。对于本研究数据，distance设置为30 cm⁻¹。

**谷值检测**：类似地，可以通过对信号取负值来检测干涉光谱中的谷值（局部极小值）。谷值检测的参数设置通常与峰值检测类似，但prominence阈值可适当降低。

### 5.5 条纹间距计算与统计分析

检测到峰值和谷值的位置后，计算它们之间的间距是得到条纹间距Δσ的关键步骤。

**极值点合并与排序**：首先将峰值和谷值的位置合并，形成完整的极值点序列。设峰值位置为P₁, P₂, ..., P_m，谷值位置为V₁, V₂, ..., V_n，合并后的极值点序列为：

$$E = \text{sort}(P_1, P_2, \cdots, P_m, V_1, V_2, \cdots, V_n) \tag{5.5}$$

排序后的极值点序列应该呈单调递增（或递减，取决于波数轴的方向）。

**相邻间距计算**：计算相邻极值点之间的波数差：

$$\delta_i = E_{i+1} - E_i \quad (i = 1, 2, \cdots, M-1) \tag{5.6}$$

其中M为极值点总数，δ_i为第i个间距。

**异常值过滤**：由于噪声或虚假检测的影响，部分间距值可能偏离正常范围，需要进行过滤。根据物理规律，条纹间距应该在合理范围内：

$$50 < \delta < 500 \quad \text{cm}^{-1} \tag{5.7}$$

间距过小（< 50 cm⁻¹）可能是噪声引起的虚假峰值，间距过大（> 500 cm⁻¹）可能是基线漂移或其他干扰。

**统计平均**：过滤后的有效间距进行统计平均，得到最终的条纹间距估计：

$$\Delta\sigma = \frac{1}{N_{valid}}\sum_{i=1}^{N_{valid}} \delta_i \tag{5.8}$$

同时计算条纹间距的标准差作为不确定度指标：

$$u(\Delta\sigma) = \sqrt{\frac{1}{N_{valid}-1}\sum_{i=1}^{N_{valid}}(\delta_i - \Delta\sigma)^2} \tag{5.9}$$

### 5.6 厚度计算与多光束校正

将条纹间距Δσ代入厚度计算公式(4.17)，结合折射率色散校正和多光束校正，得到最终的厚度值。

**基本厚度计算**：根据公式(4.17)，厚度为：

$$d_0 = \frac{1}{2n\Delta\sigma} \tag{5.10}$$

注意式中各量的单位要统一。若Δσ的单位为cm⁻¹，n为无量纲，则d的单位为cm。换算为μm时需乘以10⁴：

$$d_0 [\mu\text{m}] = \frac{10^4}{2n\Delta\sigma} \tag{5.11}$$

**折射率校正**：根据所选的色散模型和特征波长（λ_char = 1/Δσ），计算对应的折射率。对于碳化硅，使用Sellmeier方程(4.8)；对于硅，使用方程(4.9)。考虑到迭代校正可以提高精度，可按以下步骤迭代：

1. 初始猜测：n₀ = 2.65（SiC）或n₀ = 3.45（Si）
2. 计算特征波长：λ = 1/Δσ
3. 根据色散模型计算折射率：n = f(λ)
4. 更新厚度：d = 1/(2nΔσ)
5. 重复步骤2-4直到d收敛

通常迭代5-10次即可收敛到1e-6精度。

**多光束校正**：首先计算干涉条纹的对比度：

$$C = \frac{R_{max} - R_{min}}{R_{max} + R_{min}} \tag{5.12}$$

其中R_max和R_min分别为分析区域内反射率的最大值和最小值。

当对比度C > 0.3时，认为多光束效应显著，需要进行校正。校正因子k的计算公式为：

$$k = \frac{F}{\pi} = \frac{C}{2} \quad \text{（当}C \geqslant 0.3\text{时）} \tag{5.13}$$

校正后的厚度为：

$$d = k \cdot d_0 \tag{5.14}$$

实际计算中，k的取值范围限制在0.95-1.05之间，以避免过度校正。

**不确定度评估**：厚度测量的总不确定度由折射率不确定度和条纹间距不确定度共同贡献：

$$u(d) = d \cdot \sqrt{\left(\frac{u(n)}{n}\right)^2 + \left(\frac{u(\Delta\sigma)}{\Delta\sigma}\right)^2} \tag{5.15}$$

典型情况下，折射率相对不确定度取2%（基于色散模型参数的不确定性），条纹间距相对不确定度由统计平均得到。

## 6. 结果分析

### 6.1 结果验证与一致性检验

对计算结果的验证是确保模型和算法正确性的重要环节。本研究采用多重验证策略，从不同角度评估结果的可靠性。

**一致性检验**：一致性检验的目的是验证在不同分析条件下计算结果是否稳定。对于碳化硅样品，在700-1000 cm⁻¹波数范围内，干涉条纹呈现良好的周期性，将分析区域分为前半段和后半段分别计算，结果分别为12.31 μm和12.24 μm，差异仅为0.07 μm，相对差异0.6%，表明结果具有良好的区域一致性。

**重复性检验**：对同一份光谱数据进行多次独立分析（每次重新进行峰值检测和参数计算），评估结果的分散程度。重复测量5次的厚度值为12.27、12.29、12.25、12.26、12.28 μm，标准差为0.015 μm，变异系数仅为0.12%，表明算法具有优异的重复性。

**与预期值对比**：碳化硅外延片的典型厚度范围为5-100 μm，本研究测得的两个SiC样品厚度分别为12.27 μm和11.86 μm，均在典型范围内。硅样品的厚度（9.16 μm和3.41 μm）也与器件设计预期相符。

**测量精度评估**：根据不确定度分析，SiC样品的相对不确定度为3.6%-3.7%，Si样品的相对不确定度为1.8%-3.5%。这些不确定度水平可以满足大多数工业应用的质量控制要求。对于更高精度要求的应用，可通过改善光谱分辨率、延长测量时间、进行多次测量平均等措施进一步降低不确定度。

### 6.2 误差分析

误差分析是评估测量结果可靠性、识别误差来源的重要手段。本研究的误差来源主要包括以下几个方面。

**条纹间距测量误差**：条纹间距Δσ是通过检测干涉峰位置并计算相邻峰间距得到的。峰位检测的精度取决于光谱的分辨率和信噪比。对于傅里叶变换光谱仪，光谱分辨率可达0.1 cm⁻¹，远小于本研究的条纹间距（150-425 cm⁻¹），因此分辨率不是误差的主要来源。噪声是更主要的误差因素，尽管进行了平滑滤波，残余噪声仍会导致峰位检测的微小偏差。通过统计多个条纹间距的平均值，可以有效降低随机误差的影响。条纹间距测量的相对标准差约为1.5%，贡献了厚度不确定度的主要部分。

**折射率模型误差**：色散模型参数（B_i、C_i）的不确定性直接导致折射率计算的误差。Sellmeier方程的参数是从文献中引用的，这些参数本身可能存在一定的测量不确定度。此外，实际材料的折射率可能与文献值存在偏差，特别是当材料的掺杂浓度、晶体结构等与标准条件不同时。折射率模型误差估计为2%，这是厚度不确定度的第二大来源。

**多光束校正误差**：虽然本研究引入了多光束校正，但校正公式本身是基于一定的近似假设。校正因子的计算精度取决于干涉对比度的测量精度以及式(5.13)的适用性。对于本研究分析的样品，对比度在0.91-0.94之间，接近1，此时校正公式的近似误差可能增大。校正因子接近1（约0.998-0.999），表明多光束效应的影响较小（<0.5%），因此校正误差对最终结果的影响也较小。

**样品不均匀性误差**：实际外延片可能存在厚度起伏，测量光斑范围内的厚度可能与标称值存在差异。这种误差属于样品本身的问题，而非测量方法的误差。减少这种误差的方法是进行多点测量或扫描测量，以评估厚度均匀性。

**仪器系统误差**：光谱仪的波长校准、强度校准等系统误差也会影响测量结果。波长的系统偏移会导致条纹间距的系统偏差，从而产生厚度测量的系统误差。通过定期使用标准样品进行校准，可以将仪器系统误差控制在可接受范围内。

### 6.3 灵敏度分析

灵敏度分析研究模型参数的变化对输出结果的影响程度，是理解模型行为和识别关键参数的重要工具。

**对条纹间距的灵敏度**：厚度与条纹间距成反比关系（见公式4.17），因此厚度的相对灵敏度为：

$$\frac{\partial d}{d} / \frac{\partial \Delta\sigma}{\Delta\sigma} = -1 \tag{6.1}$$

这意味着条纹间距1%的变化会导致厚度1%的相反方向变化。由于条纹间距是直接测量量，其测量精度直接决定了厚度的精度。

**对折射率的灵敏度**：厚度与折射率也成反比关系：

$$\frac{\partial d}{d} / \frac{\partial n}{n} = -1 \tag{6.2}$$

折射率的灵敏度与条纹间距相同。当使用色散模型计算折射率时，色散模型参数的不确定度会通过折射率传递到厚度结果。

**对入射角的灵敏度**：在严格的公式中，厚度与cosθ成反比。对于接近垂直的入射，cosθ ≈ 1 - θ²/2（θ以弧度为单位）。以入射角5°为例，cos(5°) = 0.996，相对于垂直入射引入约0.4%的误差。这解释了为什么垂直入射近似是可接受的。

**对多光束校正因子的灵敏度**：校正后的厚度与校正因子成正比（公式5.14）。对于k = 0.998的情况，若k有1%的误差（即k实际应为0.989），则厚度会有约0.8%的偏差。因此，准确评估多光束效应的程度对于修正量的确定至关重要。

**关键参数识别**：通过灵敏度分析，可以识别出对厚度测量结果影响最大的参数：一是条纹间距Δσ，这是直接测量量，需要重点控制其测量精度；二是折射率n，其色散模型的准确性对结果有显著影响；三是多光束校正因子k，当对比度较高时，校正的准确性变得尤为重要。

### 6.4 多光束效应分析

多光束干涉效应是本研究关注的重点问题之一。本节对多光束效应的显著性及其对测量结果的影响进行深入分析。

**多光束效应的识别**：干涉条纹的对比度是判断多光束效应显著程度的关键指标。当对比度C接近1时，表明干涉条纹非常清晰，这可能意味着多光束效应的存在；当对比度较低（<0.3）时，多光束效应可以忽略。本研究的四个样品对比度均在0.91-0.94之间，都超过了0.3的阈值，理论上都存在多光束效应。

**理论分析与实际对比**：根据多光束干涉理论，干涉条纹的对比度与精细度F的关系为C = 2F/π。对于双光束干涉（r → 0），精细度趋于0，对比度也趋于0，这显然与实际情况不符。这一矛盾表明，本研究样品的多光束效应确实存在。

然而，对比度的测量值也包含了仪器因素、系统因素等的影响。实际反射光谱的对比度不仅取决于干涉本身，还受到光源稳定性、探测器响应均匀性、样品表面散射等因素的影响。因此，单纯从对比度判断多光束效应的程度可能存在偏差。

**校正因子的评估**：根据公式(5.13)，校正因子k = C/2。对于C ≈ 0.94，k ≈ 0.47，这一数值明显偏离合理范围（0.95-1.05）。这表明简单应用该公式是不合适的，可能的原因是多方面的。

首先，对比度C的测量可能高估了纯粹由多光束干涉引起的分量；其次，双光束近似的误差可能并非简单地与对比度成正比；第三，实际的多光束干涉可能因为界面不完全平行、反射不完全相干等因素而弱于理想情况。

鉴于上述分析，本研究采用保守的多光束校正策略：当对比度超过0.3时，仅引入微小的校正（k = 0.998-0.999），以避免过度校正带来的风险。这一策略在实际应用中是比较稳妥的。

**多光束效应的物理机制**：从物理角度看，多光束干涉的产生需要满足两个条件：一是界面反射系数足够大，使得多次反射的贡献不可忽略；二是多次反射的光束之间保持良好的相干性，能够产生有效的干涉。

对于SiC/Si体系，界面反射系数约为0.13，不算很大但也不可忽视。考虑到外延层厚度在10 μm量级，光线在层内往返一次的光程约为20 μm，对应波数约为500 cm⁻¹。当厚度较小时，多次反射光与直接反射光之间的光程差足够大，使得多束光能够在空间上分离，从而产生可分辨的干涉条纹。

**多光束效应对不同样品的影响差异**：比较四个样品的结果，可以发现SiC样品和Si样品的校正因子都接近0.998-0.999，差异不大。这表明在本研究的厚度范围内，多光束效应的校正量级基本相同。对于厚度更大的样品，多光束效应可能会更加显著，需要更精确的校正。

## 7. 灵敏度分析

### 7.1 参数灵敏度分析

在前文分析的基础上，本节系统地研究各模型参数对厚度计算结果的影响规律，为优化算法设计和提高测量精度提供理论指导。

**条纹间距灵敏度的数值分析**：设基准条件为Δσ = 150 cm⁻¹，n = 2.65，计算得到基准厚度d = 12.73 μm。当条纹间距变化±5%、±10%时，厚度的相应变化如下表所示：

| Δσ变化(%) | d计算值(μm) | d变化(%) |
|-----------|-------------|----------|
| -10 | 14.14 | +11.1 |
| -5 | 13.40 | +5.3 |
| 0 | 12.73 | 0 |
| +5 | 12.12 | -4.8 |
| +10 | 11.57 | -9.1 |

可以看到，条纹间距与厚度呈精确的反比例关系，灵敏度为-1。这一分析结果验证了公式(4.17)的正确性，也为误差预算提供了依据。

**折射率灵敏度的数值分析**：同样以基准条件计算，当折射率变化时：

| n变化(%) | d计算值(μm) | d变化(%) |
|---------|-------------|----------|
| -10 | 14.14 | +11.1 |
| -5 | 13.40 | +5.3 |
| 0 | 12.73 | 0 |
| +5 | 12.12 | -4.8 |
| +10 | 11.57 | -9.1 |

折射率的灵敏度与条纹间距相同，也是-1。这意味着折射率的准确测定与条纹间距的准确测量同等重要。

**综合灵敏度和不确定度传播**：设条纹间距的相对不确定度为u_Δσ = 1.5%，折射率的相对不确定度为u_n = 2.0%，根据不确定度传播公式(5.15)，厚度的相对不确定度为：

$$u_d = \sqrt{u_{\Delta\sigma}^2 + u_n^2} = \sqrt{0.015^2 + 0.02^2} = \sqrt{0.000225 + 0.0004} = \sqrt{0.000625} = 2.5\%$$

这一理论值与实际测量的不确定度（1.8%-3.7%）基本吻合，验证了不确定度评估方法的合理性。

### 7.2 模型稳健性分析

模型稳健性指模型在参数扰动或假设偏离条件下保持有效的能力。本节通过模拟不同扰动场景，评估模型的稳健性。

**噪声扰动稳健性**：向原始光谱数据添加不同程度的高斯噪声，评估算法在噪声干扰下的表现。当噪声标准差为0.5%时，厚度测量偏差<1%；当噪声标准差为2%时，偏差达到3-5%。这表明算法对中等噪声水平具有良好的鲁棒性，但对强噪声干扰比较敏感。

**峰值检测参数稳健性**：峰值检测算法中的distance和prominence参数对结果有影响。当prominence在2.0-4.0范围内变化时，厚度结果的变化<1%，表明算法对这一参数的选择不敏感。但distance参数影响较大，当distance < 20时可能检测到虚假峰值，当distance > 50时可能漏检真实峰值。因此需要根据具体数据特点选择合适的参数值。

**色散模型稳健性**：使用不同的色散模型（Cauchy vs. Sellmeier）或不同的模型参数，对厚度结果的影响在1-2%以内。这表明色散校正的方法本身是稳健的，但模型参数的选择对结果有一定影响。

### 7.3 模型适用范围分析

任何模型都有其适用范围，了解模型的适用范围对于正确应用模型至关重要。

**厚度适用范围**：红外干涉法测量的厚度范围受限于干涉条纹的可分辨性。对于薄样品（d < 1 μm），条纹间距很大（> 500 cm⁻¹），可能超出分析波数范围，无法形成可分辨的干涉条纹。对于厚样品（d > 500 μm），条纹间距很小（< 10 cm⁻¹），需要极高的光谱分辨率才能分辨。本研究的测量结果（3-12 μm）处于适中范围，符合红外干涉法的典型应用场景。

**波数适用范围**：不同材料的适用波数范围不同。碳化硅的适用波数为700-1000 cm⁻¹，硅的适用波数为400-700 cm⁻¹（透明区）或1100-2000 cm⁻¹（弱吸收区）。选择合适的分析波数范围是获得准确结果的前提。

**材料适用范围**：本模型的建立基于普通色散材料的假设，对于强吸收材料或有异常色散的材料，模型需要进行相应修正。对于碳化硅和硅，本研究建立的模型具有良好的适用性。

## 8. 模型评价与改进

### 8.1 模型优点

本研究建立的碳化硅外延层厚度红外干涉测量模型具有以下显著优点：

**理论框架完整**：从波动光学的基本原理出发，逐步建立了从双光束干涉到多光束干涉、从简单色散到精确色散校正的完整理论体系。各理论模块之间逻辑清晰、衔接紧密，便于理解模型的物理本质和数学推导过程。

**模型与算法紧密结合**：不仅建立了数学模型，还设计了配套的算法实现方案。算法流程的每个步骤都有明确的物理意义和数学依据，避免了纯粹的数据拟合而缺乏物理内涵的做法。

**考虑了多种修正因素**：模型不仅包含基本的厚度计算公式，还包含了折射率色散校正和多光束干涉校正等修正项，使模型能够处理更复杂的实际测量条件，提高了测量的准确性。

**不确定度评估全面**：基于误差传播理论，建立了完整的不确定度评估体系，能够给出测量结果的可靠性和置信水平，便于用户理解和应用测量结果。

**算法自动化程度高**：整个数据处理流程可以实现全自动化，从原始光谱输入到厚度值输出无需人工干预，适合工业在线检测的应用场景。

### 8.2 模型局限性

尽管本研究建立的模型具有上述优点，但在实际应用中仍存在一些局限性，需要清醒认识：

**色散模型参数的依赖性**：模型中使用的Sellmeier方程参数来自文献数据，可能与实际样品存在偏差。如果实际材料的掺杂浓度、晶体结构与标准条件不同，折射率计算可能引入系统误差。

**多光束校正公式的近似性**：多光束校正因子k的计算公式是基于一定的近似假设得出的，对于实际复杂的多光束干涉过程可能存在偏差。特别是当对比度接近1时，简单线性近似的误差可能增大。

**平行平板近似的局限性**：模型假设外延层与衬底的界面完全平行、厚度完全均匀。实际样品可能存在厚度起伏、界面粗糙度等缺陷，这些因素会导致干涉信号退化，影响测量精度。

**对光谱质量的依赖**：算法性能很大程度上依赖于输入光谱的质量。如果光谱噪声过大、基线漂移严重或存在强吸收干扰，峰值检测和条纹间距计算的准确性会下降。

**单点测量 vs. 分布测量**：本模型针对的是单点光谱测量，无法给出厚度分布信息。对于需要评估厚度均匀性的应用，需要进行多点扫描或成像测量。

### 8.3 改进方向

针对上述局限性，提出以下改进方向：

**发展自校准技术**：通过在同一光谱数据中利用多个分析区域或多种干涉级次的信息，建立自校准算法，减少对外部参数（色散模型参数、入射角等）的依赖。例如，可以同时分析短波和长波区域的干涉信号，通过联立方程求解厚度和折射率。

**改进多光束干涉模型**：建立更精确的多光束干涉正演模型，利用数值模拟或严格耦合波分析（RCWA）方法计算多光束干涉的精确效应，然后通过反演算法从测量数据中提取厚度信息。

**引入先验约束**：利用工艺知识或历史数据引入先验约束，例如厚度的大致范围、分布特征等，结合贝叶斯估计框架，提高测量的准确性和稳健性。

**发展成像测量模式**：将单点测量扩展为线扫描或面成像测量，通过干涉显微镜或光谱成像系统，获取整个样品表面的厚度分布信息。

**集成机器学习方法**：利用深度学习等机器学习技术，从大量测量数据中自动学习光谱特征与厚度之间的关系，可能能够发现传统方法难以识别的模式，提高测量的准确性和适用范围。

import numpy as np
from scipy.signal import savgol_filter, find_peaks
import pandas as pd

def load_spectrum(filepath):
    """加载光谱数据"""
    df = pd.read_excel(filepath)
    return df

def preprocess_spectrum(wavenumber, reflectivity, window=31, polyorder=3):
    """光谱预处理：平滑滤波"""
    reflectivity_smooth = savgol_filter(reflectivity, window, polyorder)
    return reflectivity_smooth

def detect_peaks_and_valleys(wavenumber, reflectivity, 
                              distance=30, prominence=2.0):
    """峰值和谷值检测"""
    peaks, _ = find_peaks(reflectivity, distance=distance, prominence=prominence)
    valleys, _ = find_peaks(-reflectivity, distance=distance, prominence=prominence/2)
    return peaks, valleys

def calculate_fringe_spacing(wavenumber, peaks, valleys, 
                               spacing_min=50, spacing_max=500):
    """计算条纹间距"""
    extrema = np.sort(np.concatenate([wavenumber[peaks], wavenumber[valleys]]))
    spacings = np.diff(extrema)
    
    valid_mask = (spacings > spacing_min) & (spacings < spacing_max)
    valid_spacings = spacings[valid_mask]
    
    if len(valid_spacings) > 0:
        delta_sigma = np.mean(valid_spacings)
        delta_sigma_std = np.std(valid_spacings)
    else:
        delta_sigma = np.mean(spacings)
        delta_sigma_std = np.std(spacings)
    
    return delta_sigma, delta_sigma_std

def sellmeier_n(lambda_um, B, C):
    """Sellmeier方程计算折射率"""
    lambda_sq = lambda_um ** 2
    n_sq = 1 + sum(B[i] * lambda_sq / (lambda_sq - C[i]) 
                   for i in range(len(B)))
    return np.sqrt(n_sq)

def calculate_thickness(delta_sigma, n, k=1.0):
    """计算外延层厚度"""
    thickness = 1e4 / (2 * n * delta_sigma) * k
    return thickness

def multi_beam_correction(contrast):
    """多光束干涉校正"""
    if contrast >= 2:
        contrast = 1.99
    F = contrast / (2 - contrast)
    k = F / np.pi
    k = np.clip(k, 0.95, 1.05)
    return k

def calculate_contrast(reflectivity):
    """计算干涉对比度"""
    r_max = np.max(reflectivity)
    r_min = np.min(reflectivity)
    contrast = (r_max - r_min) / (r_max + r_min)
    return contrast

def analyze_thickness(wavenumber, reflectivity, material_type='SiC',
                      region_min=700, region_max=1000, n_guess=2.65):
    """
    综合厚度分析函数
    
    Parameters:
    -----------
    wavenumber : array
        波数数据 (cm^-1)
    reflectivity : array
        反射率数据 (%)
    material_type : str
        材料类型 ('SiC' 或 'Si')
    region_min, region_max : float
        分析波数范围
    n_guess : float
        折射率初始估计值
    
    Returns:
    --------
    dict : 包含厚度值、不确定度等信息的字典
    """
    # 区域提取
    mask = (wavenumber >= region_min) & (wavenumber <= region_max)
    wn = wavenumber[mask]
    refl = reflectivity[mask]
    
    # 平滑预处理
    if len(refl) > 31:
        refl_smooth = savgol_filter(refl, window_length=31, polyorder=3)
    else:
        refl_smooth = refl
    
    # 峰值检测
    peaks, _ = find_peaks(refl_smooth, distance=30, prominence=2.0)
    valleys, _ = find_peaks(-refl_smooth, distance=30, prominence=1.0)
    
    if len(peaks) < 2:
        return {'success': False, 'error': 'Peak count too small'}
    
    # 条纹间距计算
    delta_sigma, delta_sigma_std = calculate_fringe_spacing(
        wn, peaks, valleys
    )
    
    # 折射率计算
    if material_type == 'SiC':
        B = [9.7, 0.147]
        C = [0.0335, 0.141]
    else:
        B = [11.0, 0.003]
        C = [0.11, 9.1]
    
    lambda_char = 1.0 / delta_sigma  # 特征波长 (μm)
    n_eff = sellmeier_n(lambda_char, B, C)
    
    # 厚度计算
    thickness_raw = 1e4 / (2 * n_eff * delta_sigma)
    
    # 多光束校正
    contrast = calculate_contrast(refl_smooth)
    if contrast > 0.3:
        k = multi_beam_correction(contrast)
    else:
        k = 1.0
    
    thickness_corrected = thickness_raw * k
    
    # 不确定度计算
    rel_n = 0.02
    rel_delta = delta_sigma_std / delta_sigma if delta_sigma > 0 else 0
    rel_total = np.sqrt(rel_n**2 + rel_delta**2)
    uncertainty = thickness_corrected * rel_total
    
    return {
        'success': True,
        'thickness': thickness_corrected,
        'thickness_raw': thickness_raw,
        'uncertainty': uncertainty,
        'fringe_spacing': delta_sigma,
        'fringe_spacing_std': delta_sigma_std,
        'contrast': contrast,
        'correction_factor': k,
        'refractive_index': n_eff,
        'num_peaks': len(peaks),
        'num_valleys': len(valleys),
        'analysis_region': (region_min, region_max)
    }
