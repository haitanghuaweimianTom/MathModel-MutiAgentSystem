## 二、坐标系建立与运动学模型

### 2.1 三维直角坐标系定义

建立以假目标为原点的三维直角坐标系 $O-xyz$：$x$轴正方向由假目标指向导弹初始位置的水平投影，$y$轴正方向指向真目标，$z$轴垂直向上。在此坐标系中，各关键实体位置为：

**真目标几何描述**：真目标为直立圆柱体，下底面圆心位于 $P_T = (0, 200, 0)$，其占据空间为：
$$\Omega_T = \left\{(x,y,z) \mid x^2 + (y-200)^2 \leq R^2,\ 0 \leq z \leq H\right\} \tag{1}$$

**导弹M1初始状态**：初始位置 $P_M(0) = (20000, 0, 2000)$，速度方向直指原点，速度矢量为：
$$\vec{v}_M = v_M \cdot \frac{-P_M(0)}{\|P_M(0)\|} = 300 \cdot \frac{(-20000, 0, -2000)}{\sqrt{20000^2 + 0^2 + 2000^2}} \tag{2}$$

计算得导弹速度矢量的精确表达式：
$$\vec{v}_M = \left(-\frac{300 \times 20000}{\sqrt{404000000}}, 0, -\frac{300 \times 2000}{\sqrt{404000000}}\right) = \left(-\frac{6000000}{20100}, 0, -\frac{600000}{20100}\right) \tag{3}$$

即：
$$\vec{v}_M \approx (-298.5075, 0, -29.8507)\ \text{m/s} \tag{4}$$

导弹飞行方向与水平面夹角为 $\gamma_M = \arctan(2000/20000) \approx 5.71°$，导弹到达原点所需时间：
$$t_{arrive} = \frac{\|P_M(0)\|}{v_M} = \frac{20100}{300} = 67\ \text{s} \tag{5}$$

### 2.2 无人机FY1运动模型

FY1初始位置 $P_{FY1}(0) = (17800, 0, 1800)$。无人机受领任务后瞬时调整航向，以速度 $v_{FY1} = 120$ m/s 等高度匀速直线飞行。设无人机航向在水平面内与$x$轴负方向的夹角为 $\alpha$（逆时针为正），则速度矢量为：
$$\vec{v}_{FY1} = v_{FY1}(\cos\alpha, \sin\alpha, 0) \tag{6}$$

其中 $\alpha \in [0, 2\pi)$ 为关键决策变量。无人机位置随时间演化：
$$P_{FY1}(t) = P_{FY1}(0) + \vec{v}_{FY1} \cdot t,\quad t \geq 0 \tag{7}$$

具体分量形式：
$$P_{FY1}(t) = \begin{cases} x_{FY1}(t) = 17800 + 120t\cos\alpha \\ y_{FY1}(t) = 120t\sin\alpha \\ z_{FY1}(t) = 1800 \end{cases} \tag{8}$$

## 三、干扰弹投放与烟幕云团运动模型

### 3.1 干扰弹投放点与起爆点

设干扰弹在时刻 $t_1$ 投放，投放点即为该时刻无人机位置：
$$P_{drop} = P_{FY1}(t_1) = (17800 + 120t_1\cos\alpha,\ 120t_1\sin\alpha,\ 1800) \tag{9}$$

干扰弹脱离无人机后，在重力作用下做抛物运动。投放初速度等于无人机速度 $\vec{v}_{drop} = \vec{v}_{FY1}$。弹体自由飞行时间 $\Delta t_{burst} = 3.6$ s，起爆时刻：
$$t_2 = t_1 + \Delta t_{burst} = t_1 + 3.6 \tag{10}$$

起爆点位置由弹体运动方程确定。干扰弹在重力场中的运动方程：
$$\begin{cases} \ddot{x} = 0 \\ \ddot{y} = 0 \\ \ddot{z} = -g \end{cases} \tag{11}$$

积分得速度：
$$\vec{v}_{shell}(t) = (120\cos\alpha,\ 120\sin\alpha,\ -g(t-t_1)),\quad t \in [t_1, t_2] \tag{12}$$

位置：
$$P_{shell}(t) = P_{drop} + \int_{t_1}^{t} \vec{v}_{shell}(\tau)\, d\tau \tag{13}$$

起爆点 $P_{burst} = P_{shell}(t_2)$ 的各分量为：
$$\begin{cases} x_{burst} = 17800 + 120t_1\cos\alpha + 120 \times 3.6 \times \cos\alpha = 17800 + 120(t_1+3.6)\cos\alpha \\ y_{burst} = 120t_1\sin\alpha + 120 \times 3.6 \times \sin\alpha = 120(t_1+3.6)\sin\alpha \\ z_{burst} = 1800 - \frac{1}{2}g(3.6)^2 = 1800 - 63.504 = 1736.496 \end{cases} \tag{14}$$

### 3.2 烟幕云团中心运动

烟幕云团起爆后瞬时形成球状烟幕，中心以 $v_{sink} = 3$ m/s 匀速下沉。云团中心位置：
$$P_{smoke}(t) = \begin{cases} P_{burst}, & t < t_2 \\ (x_{burst},\ y_{burst},\ z_{burst} - v_{sink}(t-t_2)), & t \in [t_2, t_2+T_{smoke}] \\ \text{失效}, & t > t_2 + T_{smoke} \end{cases} \tag{15}$$

烟幕有效时间窗口为 $[t_2, t_2 + 20]$，在此区间内有效遮蔽区域为以 $P_{smoke}(t)$ 为球心、半径 $r_{eff} = 10$ m 的固定球体：
$$\mathcal{B}(t) = \left\{ P \in \mathbb{R}^3 \mid \|P - P_{smoke}(t)\| \leq r_{eff} \right\} \tag{16}$$

## 四、视线穿透判定模型与有效遮蔽时长计算

### 4.1 导弹-目标连线方程

有效遮蔽的本质是：烟幕云团阻挡了导弹对真目标的视线。定义导弹位置 $P_M(t)$ 与真目标参考点 $P_T = (0, 200, 0)$ 的连线为**威胁视线**。为简化分析，取真目标中心点 $P_T$ 作为被保护参考点（因真目标尺度远小于导弹距离，该近似合理）。

导弹位置随时间演化：
$$P_M(t) = P_M(0) + \vec{v}_M \cdot t = (20000 - 298.5075t,\ 0,\ 2000 - 29.8507t) \tag{17}$$

威胁视线参数方程：对任意时刻 $t$，从导弹指向目标的射线为：
$$L(s; t) = P_M(t) + s \cdot \vec{d}(t),\quad s \in [0, \|P_M(t)-P_T\|] \tag{18}$$

其中方向向量：
$$\vec{d}(t) = P_T - P_M(t) = (-20000 + 298.5075t,\ 200,\ -2000 + 29.8507t) \tag{19}$$

### 4.2 视线-烟幕球体相交判定

对于烟幕有效时间窗口内的任意时刻 $t \in [t_2, t_2+20]$，计算威胁视线与烟幕球体 $\mathcal{B}(t)$ 的相交情况。将视线表示为：
$$\vec{r}(s) = P_M(t) + s \cdot \frac{\vec{d}(t)}{\|\vec{d}(t)\|} \tag{20}$$

或采用更高效的参数形式，直接求解线段 $P_M(t)P_T$ 与球体 $\mathcal{B}(t)$ 的交点。定义向量：
$$\vec{w}(t) = P_M(t) - P_{smoke}(t) \tag{21}$$

线段上点到球心距离平方的最小值决定相交性。令线段参数 $s \in [0,1]$：
$$\vec{r}(s) = P_M(t) + s(P_T - P_M(t)) \tag{22}$$

到球心距离平方：
$$f(s) = \|\vec{r}(s) - P_{smoke}(t)\|^2 = \|\vec{w}(t) + s(P_T-P_M(t))\|^2 \tag{23}$$

这是关于 $s$ 的二次函数。设 $\vec{u}(t) = P_T - P_M(t)$，则：
$$f(s) = \|\vec{u}\|^2 s^2 + 2(\vec{w}\cdot\vec{u})s + \|\vec{w}\|^2 \tag{24}$$

最小值在 $s^* = -\frac{\vec{w}\cdot\vec{u}}{\|\vec{u}\|^2}$ 处取得。视线穿透烟幕的充要条件为：
$$\begin{cases} 0 \leq s^* \leq 1 & \text{（最近点在线段上）} \\ f(s^*) \leq r_{eff}^2 = 100 & \text{（最近点进入球体）} \end{cases} \tag{25}$$

等价地，定义判别函数：
$$\delta(t; t_1, \alpha) = \min_{s \in [0,1]} \|\vec{r}(s) - P_{smoke}(t)\|^2 - r_{eff}^2 \tag{26}$$

则时刻 $t$ 存在有效遮蔽当且仅当 $\delta(t; t_1, \alpha) \leq 0$。

### 4.3 有效遮蔽时长计算

有效遮蔽时长为威胁视线被烟幕阻挡的时间测度：
$$T_{eff}(t_1, \alpha) = \mu\left(\left\{ t \in [t_2, t_2+20] \cap [0, t_{arrive}] \mid \delta(t; t_1, \alpha) \leq 0 \right\}\right) \tag{27}$$

其中 $\mu(\cdot)$ 为勒贝格测度。由于导弹约67s到达原点，而烟幕有效仅20s，需优化使烟幕窗口与导弹威胁窗口匹配。

# FY1单弹干扰M1优化策略的数学模型

## 一、问题分析与坐标系建立

### 1.1 导弹轨迹的精确确定

依据题目给定条件，导弹M1初始位置为$\mathbf{P}_{M1}^0 = (20000, 0, 2000)$，飞行速度$v_M = 300$ m/s，飞行方向直指假目标原点$(0,0,0)$。由此可唯一确定M1的运动轨迹。

导弹飞行方向的单位向量为：
$$\hat{d}_{M1} = \frac{(0-20000, 0-0, 0-2000)}{\|(-20000, 0, -2000)\|} = \frac{(-20000, 0, -2000)}{\sqrt{20000^2 + 2000^2}} = \frac{(-10, 0, -1)}{\sqrt{101}} \tag{1}$$

因此M1的位置随时间演化方程为：
$$\mathbf{P}_{M1}(t) = (20000, 0, 2000) + 300t \cdot \hat{d}_{M1} = \left(20000 - \frac{3000t}{\sqrt{101}},\ 0,\ 2000 - \frac{300t}{\sqrt{101}}\right) \tag{2}$$

引入参数简化表示，令$a = \frac{300}{\sqrt{101}} \approx 29.851$ m/s，则：
$$\mathbf{P}_{M1}(t) = (20000 - 10at,\ 0,\ 2000 - at) \tag{3}$$

导弹到达假目标原点的时间为$T_{M1} = \frac{\sqrt{101} \times 20000}{3000} = \frac{20\sqrt{101}}{3} \approx 66.89$ s。

### 1.2 关键几何观察：降维分析

由于M1的轨迹位于$xOz$平面（$y \equiv 0$），且真目标位置为$(0, 200, 0)$，云团中心垂直下沉（仅有$z$向运动），可将三维遮蔽问题降维分析。

定义**导弹轨迹直线**$L_{M1}$：过点$(20000, 0, 2000)$，方向向量$(-10, 0, -1)$，即：
$$L_{M1}: \frac{x-20000}{-10} = \frac{z-2000}{-1},\ y = 0 \tag{4}$$

或参数形式：$(x, z) = (20000 - 10s,\ 2000 - s)$，$s \geq 0$。

## 二、运动学约束与起爆点计算

### 2.1 无人机运动方程

FY1以恒定高度$h = 1800$ m匀速直线飞行：
$$\mathbf{P}_{FY1}(t) = (17800 + v\cos\theta \cdot t,\ v\sin\theta \cdot t,\ 1800), \quad t \geq 0 \tag{5}$$

### 2.2 干扰弹抛体运动与起爆点

干扰弹于$t_1$时刻投放，初速度继承无人机状态$\mathbf{v}_0 = (v\cos\theta, v\sin\theta, 0)$，受重力作用运动。在$t_2 = t_1 + \Delta t$时刻起爆，起爆点位置为：
$$\mathbf{P}_{burst} = \mathbf{P}_{FY1}(t_1) + \mathbf{v}_0 \cdot \Delta t + \frac{1}{2}\mathbf{g}(\Delta t)^2 \tag{6}$$

展开得分量形式：
$$x_b = 17800 + v\cos\theta \cdot t_1 + v\cos\theta \cdot \Delta t = 17800 + v\cos\theta \cdot t_2 \tag{7}$$
$$y_b = v\sin\theta \cdot t_1 + v\sin\theta \cdot \Delta t = v\sin\theta \cdot t_2 \tag{8}$$
$$z_b = 1800 - \frac{g(\Delta t)^2}{2} \tag{9}$$

**关键约束**：起爆点高度需满足$z_b > 0$（地面约束），且为形成有效遮蔽，通常要求$z_b \geq R_{eff} = 10$ m。

### 2.3 烟幕云团中心轨迹

$$\mathbf{P}_{cloud}(t) = (x_b,\ y_b,\ z_b - 3(t-t_2)), \quad t \in [t_2, t_2 + 20] \tag{10}$$

## 三、遮蔽判定与相交时间计算

### 3.1 视线-球相交判定

在任意时刻$t$，导弹M1位置为$\mathbf{P}_{M1}(t)$，其导引头视线指向真目标$\mathbf{P}_{tgt} = (0, 200, 0)$。视线线段参数方程为：
$$\mathbf{L}(s; t) = \mathbf{P}_{M1}(t) + s \cdot (\mathbf{P}_{tgt} - \mathbf{P}_{M1}(t)), \quad s \in [0, 1] \tag{11}$$

云团有效遮蔽条件：视线线段与以$\mathbf{P}_{cloud}(t)$为中心、半径$R_{eff} = 10$ m的球体存在交点。

定义向量$\mathbf{w}(t) = \mathbf{P}_{M1}(t) - \mathbf{P}_{cloud}(t)$，视线方向$\mathbf{u}(t) = \mathbf{P}_{tgt} - \mathbf{P}_{M1}(t)$。视线到球心距离为：
$$d(t) = \frac{\|\mathbf{w}(t) \times \mathbf{u}(t)\|}{\|\mathbf{u}(t)\|} \tag{12}$$

遮蔽判定条件为：
$$d(t) \leq R_{eff} = 10 \tag{13}$$

且交点需位于视线线段上（即存在$s \in [0,1]$使$\|\mathbf{L}(s;t) - \mathbf{P}_{cloud}(t)\| \leq R_{eff}$）。

### 3.2 降维简化：导弹轨迹参数化

利用M1轨迹位于$y=0$平面的特性，引入导弹轨迹参数$\tau \in [0, T_{M1}]$表示时间。M1位置：
$$\mathbf{P}_{M1}(\tau) = (20000 - 10a\tau,\ 0,\ 2000 - a\tau) \tag{14}$$

对于云团中心$(x_b, y_b, z_b - 3(t-t_2))$，由于M1的$y \equiv 0$，定义**有效距离**：
$$D_{eff}(\tau, t) = \sqrt{(x_{M1}(\tau) - x_b)^2 + y_b^2 + (z_{M1}(\tau) - z_b + 3(t-t_2))^2} \tag{15}$$

但更准确的做法是计算视线-球相交。由于真目标位于$(0,200,0)$，而M1轨迹在$y=0$平面，视线方向具有$y$分量，这是三维问题的本质特征。

### 3.3 单弹遮蔽时间积分

定义指示函数：
$$\mathcal{I}(t; \mathbf{P}_{burst}, t_2) = \begin{cases} 1, & d(t) \leq R_{eff} \text{ 且视线线段与球相交} \\ 0, & \text{否则} \end{cases} \tag{16}$$

单弹有效遮蔽时间为：
$$T_{cover} = \int_{t_2}^{t_2+T_{eff}} \mathcal{I}(t; \mathbf{P}_{burst}, t_2)\, dt \tag{17}$$

优化目标为最大化$T_{cover}$：
$$\max_{v, \theta, t_1, t_2}\ T_{cover}(v, \theta, t_1, t_2) \tag{18}$$

## 四、结构化求解策略

### 4.1 几何分析：最优遮蔽位置特征

为使遮蔽时间最大化，需分析云团在有效时间窗口$[t_2, t_2+20]$内的运动与导弹视线的匹配关系。

**命题1（最优性必要条件）**：若云团中心轨迹与导弹视线存在持续相交，则云团下沉运动应与导弹高度下降相匹配。

导弹M1的$z$坐标变化率：$\dot{z}_M = -\frac{300}{\sqrt{101}} \approx -29.85$ m/s。

云团下沉速度仅为3 m/s，远小于导弹高度变化率。因此，**云团难以通过垂直运动持续跟踪导弹视线**，必须依靠水平位置的精确配置，使导弹在穿越云团附近时获得最大遮蔽时间。

### 4.2 关键截面分析

定义**威胁平面**：过真目标$(0,200,0)$且垂直于M1初始视线的平面。但更实用的方法是考虑导弹飞行路径上的关键区域。

由于云团半径仅10 m，而M1速度300 m/s，若云团静止，导弹穿越云团时间仅约$2 \times 10 / 300 \approx 0.067$ s。因此，**必须利用云团运动与导弹视线的相对几何关系**。

实际上，遮蔽判定是视线-球相交而非弹体-球相交。视线长度为导弹到真目标的距离，随时间从约20000 m递减至200 m（真目标处）。当导弹远离时，视线很长，云团易与之相交；当导弹接近时，视线变短，相交条件更严格。

### 4.3 优化变量降维

由式(7)-(9)，起爆点$(x_b, y_b, z_b)$由$(v, \theta, t_2, \Delta t)$决定。但注意到：
- $x_b$和$y_b$仅依赖于$v, \theta, t_2$的乘积组合，即无人机在$t_2$时刻的水平位置
- $z_b$仅依赖于$\Delta t = t_2 - t_1$

因此可重新参数化：以起爆点坐标$(x_b, y_b, z_b)$为中间变量，再反解控制参数。

**起爆点可行域约束**：
- 水平可达性：存在$v \in [70,140]$，$\theta \in [0,2\pi)$，$t_2 \geq 0$使得$(x_b, y_b) = (17800 + v\cos\theta \cdot t_2,\ v\sin\theta \cdot t_2)$
- 即$(x_b - 17800)^2 + y_b^2 \in [70t_2, 140t_2]^2$，且$z_b = 1800 - g(\Delta t)^2/2 \geq 0$

### 4.4 单弹最大遮蔽时间上界分析

考虑理想情况：云团中心始终位于导弹视线上，且视线穿过球心。此时遮蔽时间受限于：
- 云团有效持续时间：20 s
- 导弹穿越云球时间：因视线长度变化，非恒定

实际上，由于云团以3 m/s下沉，而视线方向变化，持续20 s完全遮蔽几乎不可能。需数值优化求解。

### 4.5 数值优化框架

对于给定起爆点$\mathbf{P}_{burst}$和起爆时刻$t_2$，计算遮蔽时间$T_{cover}$的算法：

**步骤1**：对$t \in [t_2, t_2+20]$离散化（步长$\delta t = 0.01$ s）

**步骤2**：对每个$t$：
- 计算$\mathbf{P}_{M1}(t)$，$\mathbf{P}_{cloud}(t) = (x_b, y_b, z_b - 3(t-t_2))$
- 计算视线方向$\mathbf{u} = \mathbf{P}_{tgt} - \mathbf{P}_{M1}(t)$
- 计算$\mathbf{w} = \mathbf{P}_{M1}(t) - \mathbf{P}_{cloud}(t)$
- 计算距离$d = \|\mathbf{w} \times \mathbf{u}\| / \|\mathbf{u}\|$
- 若$d \leq 10$，进一步检验交点是否在线段$[0,1]$内

**步骤3**：累计满足条件的总时间

外层采用全局优化算法（如遗传算法、粒子群或网格搜索+局部优化）求解：
$$\max_{x_b, y_b, z_b, t_2}\ T_{cover}(x_b, y_b, z_b, t_2) \tag{19}$$

### 二、M1导弹轨迹的显式参数化

#### 2.1 M1运动学方程

M1初始位置 $\mathbf{P}_{M1}^0 = (20000, 0, 2000)$，速度 $v_M = 300$ m/s，飞行方向直指假目标原点 $(0,0,0)$。其运动方向单位向量为：

$$\hat{d}_{M1} = \frac{(0-20000, 0-0, 0-2000)}{\sqrt{20000^2 + 0^2 + 2000^2}} = \frac{(-20000, 0, -2000)}{\sqrt{404000000}} = \frac{(-10, 0, -1)}{\sqrt{101}} \tag{1}$$

M1位置随时间的显式表达式为：

$$\mathbf{P}_{M1}(t) = \mathbf{P}_{M1}^0 + v_M \cdot t \cdot \hat{d}_{M1} = \left(20000 - \frac{3000t}{\sqrt{101}},\ 0,\ 2000 - \frac{300t}{\sqrt{101}}\right) \tag{2}$$

引入简化记号 $c = \frac{300}{\sqrt{101}} \approx 29.8515$ m/s，则：

$$\mathbf{P}_{M1}(t) = \left(20000 - 10ct,\ 0,\ 2000 - ct\right), \quad t \in [0, T_{M1}^{total}] \tag{3}$$

其中总飞行时间 $T_{M1}^{total} = \frac{\sqrt{404000000}}{300} \approx 67.15$ s。

#### 2.2 M1与真目标的几何关系

真目标为圆柱体：下底面圆心 $(0, 200, 0)$，半径 $r_{tgt} = 7$ m，高度 $h_{tgt} = 10$ m。真目标在水平面上的投影为圆盘 $\mathcal{D}_{tgt} = \{(x,y,0) : x^2 + (y-200)^2 \leq 49\}$。

M1发现真目标的临界条件需分析视线与真目标的几何关系。设M1在时刻 $t$ 的位置为 $\mathbf{P}_{M1}(t) = (x_M(t), 0, z_M(t))$，其"发现真目标"等价于存在真目标表面点 $\mathbf{Q}$ 使得：
- 视线向量 $\mathbf{P}_{M1}(t) \to \mathbf{Q}$ 不被烟幕遮挡
- 该视线与真目标表面相交

由于M1轨迹在 $y=0$ 平面内，而真目标中心在 $(0,200,0)$，两者存在水平偏移。M1与真目标中心连线的水平距离为：

$$d_{horiz}(t) = \sqrt{x_M(t)^2 + 200^2} \tag{4}$$

M1能够"发现"真目标的关键时段是当其与真目标的水平距离足够近、且俯仰角度允许目视时。具体而言，定义**可发现时段** $\mathcal{T}_{discover} \subset [0, T_{M1}^{total}]$ 为满足以下条件的时刻集合：

$$\mathcal{T}_{discover} = \left\{t \geq 0 : d_{horiz}(t) \leq D_{max}^{sensor},\ z_M(t) \geq z_{min}^{sensor}\right\} \tag{5}$$

其中 $D_{max}^{sensor}$ 为导弹传感器最大作用距离，$z_{min}^{sensor}$ 为最低工作高度。若题目未明确给出，可保守取M1从发现真目标几何轮廓至飞临真目标上方的时段。经计算，M1在 $x_M(t) \approx 0$ 即 $t \approx 67$ s 时最接近真目标水平位置，但此时高度已很低。更关键的是M1在飞行中后段，当水平距离进入数千米范围时的发现概率。

为精确刻画，定义M1与真目标表面最近点的距离：

$$d_{M1\to tgt}(t) = \min_{\mathbf{Q} \in \text{真目标}} \|\mathbf{P}_{M1}(t) - \mathbf{Q}\| \tag{6}$$

由于真目标为圆柱体，该距离需分情况讨论。设M1在水平面投影为 $(x_M(t), 0)$，真目标中心投影为 $(0, 200)$：

**情形A**：若 $(x_M(t), 0)$ 到圆心 $(0,200)$ 的水平距离 $\rho(t) = \sqrt{x_M(t)^2 + 40000} > 7$ m，则最近点在圆柱侧面或顶面/底面边缘。

**情形B**：若 $\rho(t) \leq 7$ m，则M1在真目标正上方或正下方，最近点为顶面或底面中心。

对于本题，由于真目标半径仅7m而水平偏移200m，$\rho(t) \geq 200 - 7 = 193$ m 恒成立，故始终为情形A。此时圆柱侧面最近点的水平投影为圆心连线与圆周的交点。

定义水平投影夹角：$\cos\theta(t) = \frac{-x_M(t)}{\rho(t)}$，$\sin\theta(t) = \frac{200}{\rho(t)}$（取朝向M1的方向）。

圆柱侧面最近点 $\mathbf{Q}_{side} = (-7\cos\theta(t),\ 200 - 7\sin\theta(t),\ z_Q)$，其中 $z_Q \in [0, 10]$ 使距离最小。

M1到真目标的距离函数为：

$$d_{M1\to tgt}(t) = \min\left\{\sqrt{(\rho(t)-7)^2 + (z_M(t)-z_Q^*)^2},\ \sqrt{\rho(t)^2 + z_M(t)^2},\ \sqrt{\rho(t)^2 + (z_M(t)-10)^2}\right\} \tag{7}$$

分别对应侧面、底面、顶面最近点，其中 $z_Q^* = \text{clip}(z_M(t), [0,10])$。

### 三、弹体运动学与云团演化

#### 3.1 无人机平台运动

FY1以恒定高度、恒定速度沿直线飞行：

$$\mathbf{P}_{FY1}(t) = \mathbf{P}_{FY1}^0 + v \cdot t \cdot \hat{n} = (17800 + vn_x t,\ vn_y t,\ 1800), \quad t \geq 0 \tag{8}$$

#### 3.2 弹体投放后运动

第$k$枚弹在 $t_1^{(k)}$ 时刻于投放点 $\mathbf{P}_{drop}^{(k)} = \mathbf{P}_{FY1}(t_1^{(k)})$ 脱离无人机。弹体脱离后仅受重力，初速度为无人机投放时刻速度 $\mathbf{V}_{drop}^{(k)} = v\hat{n} = (vn_x, vn_y, 0)$。

弹体运动分解为水平匀速与竖直自由落体：

$$\mathbf{P}_{shell}^{(k)}(t) = \begin{cases} \mathbf{P}_{drop}^{(k)}, & t < t_1^{(k)} \\ \left(x_{drop}^{(k)} + vn_x(t-t_1^{(k)}),\ y_{drop}^{(k)} + vn_y(t-t_1^{(k)}),\ z_0 - \frac{g(t-t_1^{(k)})^2}{2}\right), & t \geq t_1^{(k)} \end{cases} \tag{9}$$

#### 3.3 起爆点约束

起爆时刻 $t_2^{(k)}$ 需满足时间引信约束，即弹体在 $t_2^{(k)}$ 时刻到达起爆点：

$$\mathbf{P}_{burst}^{(k)} = \mathbf{P}_{shell}^{(k)}(t_2^{(k)}) = \left(x_{drop}^{(k)} + vn_x\Delta t^{(k)},\ y_{drop}^{(k)} + vn_y\Delta t^{(k)},\ z_0 - \frac{g(\Delta t^{(k)})^2}{2}\right) \tag{10}$$

其中 $\Delta t^{(k)} = t_2^{(k)} - t_1^{(k)} > 0$ 为弹体飞行时间。

**关键约束**：起爆点高度必须非负（地面约束），且弹体飞行时间需合理：

$$z_{burst}^{(k)} = z_0 - \frac{g(\Delta t^{(k)})^2}{2} \geq 0 \quad \Rightarrow \quad \Delta t^{(k)} \leq \sqrt{\frac{2z_0}{g}} = \sqrt{\frac{3600}{9.8}} \approx 19.18 \text{ s} \tag{11}$$

#### 3.4 云团运动与有效时段

云团起爆后瞬时形成球状烟幕，中心以 $v_{sink} = 3$ m/s 匀速下沉，有效半径固定 $R_{eff} = 10$ m，有效寿命 $T_{life} = 20$ s。

云团中心轨迹：

$$\mathbf{P}_{cloud}^{(k)}(t) = \begin{cases} \text{未起爆}, & t < t_2^{(k)} \\ \left(x_{burst}^{(k)},\ y_{burst}^{(k)},\ z_{burst}^{(k)} - v_{sink}(t-t_2^{(k)})\right), & t \in [t_2^{(k)},\ t_2^{(k)}+T_{life}] \\ \text{失效}, & t > t_2^{(k)}+T_{life} \end{cases} \tag{12}$$

云团有效存在时段为 $[t_2^{(k)},\ t_2^{(k)}+T_{life}]$。

### 四、遮蔽判定的几何拦截模型

#### 4.1 视线遮蔽条件

M1在时刻 $t$ 对真目标的视线被第$k$枚云团遮蔽，当且仅当：
- $t \in [t_2^{(k)},\ t_2^{(k)}+T_{life}]$（云团有效）
- M1与真目标表面某点的连线穿过云团球体

等价地，存在真目标表面点 $\mathbf{Q}$，使得线段 $\mathbf{P}_{M1}(t)\mathbf{Q}$ 与以 $\mathbf{P}_{cloud}^{(k)}(t)$ 为中心、半径 $R_{eff}$ 的球体相交。

为简化分析，采用**保守遮蔽判定**：若M1到云团中心的距离不超过 $R_{eff}$，则M1位于云团内部或表面，此时若M1朝向真目标的视线经过云团核心区域，判定为有效遮蔽。更精确地，定义**视线-云团相交检测**：

设M1在时刻 $t$ 的位置为 $\mathbf{M} = \mathbf{P}_{M1}(t)$，云团中心为 $\mathbf{C} = \mathbf{P}_{cloud}^{(k)}(t)$。M1朝向真目标的视线方向为 $\hat{l} = \frac{\mathbf{Q}_{nearest} - \mathbf{M}}{\|\mathbf{Q}_{nearest} - \mathbf{M}\|}$，其中 $\mathbf{Q}_{nearest}$ 为真目标上距M1最近点。

视线参数方程：$\mathbf{r}(s) = \mathbf{M} + s\hat{l},\ s \geq 0$。

该射线与云团球面的交点满足 $\|\mathbf{r}(s) - \mathbf{C}\|^2 = R_{eff}^2$，即：

$$\|\mathbf{M} + s\hat{l} - \mathbf{C}\|^2 = R_{eff}^2 \tag{13}$$

展开得二次方程：

$$s^2 + 2s\hat{l}\cdot(\mathbf{M}-\mathbf{C}) + \|\mathbf{M}-\mathbf{C}\|^2 - R_{eff}^2 = 0 \tag{14}$$

判别式 $\Delta = 4[\hat{l}\cdot(\mathbf{M}-\mathbf{C})]^2 - 4[\|\mathbf{M}-\mathbf{C}\|^2 - R_{eff}^2]$。

遮蔽发生的充要条件为：
- $\Delta \geq 0$（有实交点）
- 存在正根 $s^* > 0$ 使得 $\mathbf{r}(s^*)$ 位于M1与真目标之间

由于M1飞向真目标方向，$s > 0$ 自动满足。进一步要求交点在真目标之前，即 $s^* < \|\mathbf{Q}_{nearest} - \mathbf{M}\|$。

#### 4.2 一维时间约束简化

上述三维几何检测可转化为关于时间 $t$ 的一维判定函数。定义第$k$枚弹对M1的**瞬时遮蔽指示函数**：

## 三、无人机分段运动学模型

### 3.1 严格分段运动学

无人机运动需明确区分投弹前、投弹后两个阶段：

$$\mathbf{P}_{FYi}(t) = \begin{cases} \mathbf{P}_{FYi}^0 + v_i t \cdot \hat{n}_i, & 0 \leq t \leq t_{1,i}^{(1)} \\ \mathbf{P}_{FYi}^0 + v_i t_{1,i}^{(1)} \cdot \hat{n}_i + v_i (t - t_{1,i}^{(1)}) \cdot \hat{n}_i, & t > t_{1,i}^{(1)} \end{cases} \tag{1}$$

简化后统一表达式：

$$\mathbf{P}_{FYi}(t) = \mathbf{P}_{FYi}^0 + v_i \cdot t \cdot \hat{n}_i, \quad \forall t \geq 0 \tag{2}$$

式(2)表明无人机始终以原速度、原方向飞行，投弹动作不改变其运动状态，仅释放弹体。

### 3.2 投放点与弹体初状态

$$\mathbf{P}_{drop,i}^{(1)} = \mathbf{P}_{FYi}^0 + v_i t_{1,i}^{(1)} \hat{n}_i \tag{3}$$

$$\dot{\mathbf{P}}_{drop,i}^{(1)} = v_i \hat{n}_i = (v_i n_{x,i}, v_i n_{y,i}, 0) \tag{4}$$

弹体初速度继承无人机水平速度，垂直初速度为零。

## 五、视线阻断模型

### 5.1 真目标几何描述

真目标为圆柱体，下底面圆心 $(0, 200, 0)$，半径 $R_{tgt}=7$ m，高 $H_{tgt}=10$ m。圆柱表面参数方程：

$$\mathcal{T} = \left\{(x,y,z) \mid x^2 + (y-200)^2 \leq 49, \; 0 \leq z \leq 10\right\} \tag{11}$$

### 5.2 M1与真目标的视线方程

M1位置 $\mathbf{M1}(t) = (20000-300t, 0, 2000)$。对于任意时刻 $t$，M1至真目标圆柱体上任意点 $\mathbf{Q} \in \partial\mathcal{T}$ 的视线为：

$$\mathcal{L}(t, \mathbf{Q}) = \left\{\mathbf{M1}(t) + s(\mathbf{Q} - \mathbf{M1}(t)) \mid s \in [0,1]\right\} \tag{12}$$

### 5.3 视线阻断判定

烟幕云团 $i$ 在时刻 $t$ 形成球体 $\mathcal{S}_i^{(1)}(t) = \{\mathbf{P} \mid \|\mathbf{P} - \mathbf{C}_i^{(1)}(t)\| \leq R_{eff}\}$。

**定义（视线阻断）**：对于时刻 $t$，若存在烟幕云团 $i$ 使得视线 $\mathcal{L}(t, \mathbf{Q})$ 与球体 $\mathcal{S}_i^{(1)}(t)$ 相交，则称该视线被阻断。即：

$$\exists i \in \{1,2,3\}, \exists s \in [0,1]: \|\mathbf{M1}(t) + s(\mathbf{Q} - \mathbf{M1}(t)) - \mathbf{C}_i^{(1)}(t)\| \leq R_{eff} \tag{13}$$

### 5.4 视线-球体相交解析条件

将视线参数化代入球面方程，令 $\mathbf{d} = \mathbf{Q} - \mathbf{M1}(t)$，$\mathbf{w} = \mathbf{M1}(t) - \mathbf{C}_i^{(1)}(t)$，得二次方程：

$$\|\mathbf{d}\|^2 s^2 + 2(\mathbf{d} \cdot \mathbf{w})s + \|\mathbf{w}\|^2 - R_{eff}^2 = 0 \tag{14}$$

判别式 $\Delta = 4(\mathbf{d}\cdot\mathbf{w})^2 - 4\|\mathbf{d}\|^2(\|\mathbf{w}\|^2 - R_{eff}^2)$。当 $\Delta \geq 0$ 时，需进一步检验根 $s^* = \frac{-(\mathbf{d}\cdot\mathbf{w}) \pm \sqrt{\Delta/4}}{\|\mathbf{d}\|^2}$ 是否满足 $s^* \in [0,1]$。

### 5.5 全目标遮蔽约束

为确保M1完全无法发现真目标，需对圆柱体表面采样或采用保守估计。考虑圆柱体轴向特征点集 $\mathcal{Q}_{sample} = \{(0,200,0), (0,200,10), (7,200,5), (-7,200,5), (0,193,5), (0,207,5)\}$，定义遮蔽指示函数：

$$\mathcal{I}_{block}(t) = \bigwedge_{\mathbf{Q} \in \mathcal{Q}_{sample}} \left[\bigvee_{i=1}^{3} \mathbb{1}_{\mathcal{L}(t,\mathbf{Q}) \cap \mathcal{S}_i^{(1)}(t) \neq \emptyset}\right] \tag{15}$$

当 $\mathcal{I}_{block}(t) = 1$ 时，认为时刻 $t$ 实现有效遮蔽。

## 六、弹体可达域分析

### 6.1 可达域定义

第 $i$ 架无人机的弹体可达域 $\mathcal{R}_i$ 为：该无人机在所有允许速度、方向下，经任意投弹时刻、起爆延迟组合，弹体起爆点可达到的空间区域。

### 6.2 可达域边界推导

由式(7)-(9)，起爆点满足：

$$x_{Bi}^{(1)} = x_{FYi}^0 + v_i (t_{1,i}^{(1)} + \Delta t_i^{(1)}) n_{x,i} \tag{16}$$

$$y_{Bi}^{(1)} = y_{FYi}^0 + v_i (t_{1,i}^{(1)} + \Delta t_i^{(1)}) n_{y,i} \tag{17}$$

$$z_{Bi}^{(1)} = z_{FYi}^0 - \frac{g}{2}(\Delta t_i^{(1)})^2 \tag{18}$$

$$(x_{Bi}^{(1)} - x_{FYi}^0)^2 + (y_{Bi}^{(1)} - y_{FYi}^0)^2 = D_i^2 \tag{19}$$

$$z_{Bi}^{(1)} = z_{FYi}^0 - \frac{g}{2}\left(T_i - t_{1,i}^{(1)}\right)^2 \leq z_{FYi}^0 \tag{20}$$

结合物理实际，要求起爆高度 $z_{Bi}^{(1)} \geq 0$（地面约束），则：

$$\Delta t_i^{(1)} \leq \sqrt{\frac{2z_{FYi}^0}{g}} \tag{21}$$

- FY1: $\Delta t_{max,1} = \sqrt{2 \times 1800 / 9.8} \approx 19.15$ s
- FY2: $\Delta t_{max,2} = \sqrt{2 \times 1400 / 9.8} \approx 16.90$ s
- FY3: $\Delta t_{max,3} = \sqrt{2 \times 700 / 9.8} \approx 11.95$ s

M1威胁轴为 $y=0, z=2000$ 的直线（水平高度2000 m）。由于 $z_{FYi}^0 < 2000$ 且弹体仅下沉不上升，**弹体起爆点无法达到 $z=2000$ m 高度**。因此烟幕云团需通过水平漂移（无人机飞行方向控制）在M1飞行路径前方形成遮蔽，而非直接在M1高度层起爆。

**关键结论**：三架无人机均无法在M1飞行高度层（2000 m）直接起爆形成云团。烟幕遮蔽必须依赖云团水平范围的覆盖，即M1视线穿过云团时，云团中心与M1存在显著高度差，但水平距离在10 m以内。

## 二、烟幕干扰弹弹道模型

### 2.1 投放点与起爆点的运动学关系

设第$k$枚弹由第$j(k)$架无人机在$t_{0,k}$时刻于投放点$\mathbf{P}_k^{drop} = (x_k^{drop}, y_k^{drop}, z_k^{drop})$释放，于$t_{1,k}$时刻在起爆点$\mathbf{P}_k^{det} = (x_k^{det}, y_k^{det}, z_k^{det})$起爆。投放间隔：
$$\Delta t_k = t_{1,k} - t_{0,k} > 0 \tag{2}$$

干扰弹脱离无人机后仅受重力作用，水平方向保持投放时的速度（即无人机速度），竖直方向受重力加速。设无人机飞行高度不变（$z_j^{drop} = z_j^0$），则：

**水平方向**（匀速运动）：
$$x_k^{det} = x_k^{drop} + v_{j(k)} n_{j(k),x} \cdot \Delta t_k \tag{3}$$
$$y_k^{det} = y_k^{drop} + v_{j(k)} n_{j(k),y} \cdot \Delta t_k \tag{4}$$

**竖直方向**（自由落体）：
$$z_k^{det} = z_k^{drop} - \frac{g}{2}(\Delta t_k)^2 \tag{5}$$

**关键约束**：投放点与起爆点不能等同，即$\Delta t_k > 0$且至少满足：
$$\|\mathbf{P}_k^{det} - \mathbf{P}_k^{drop}\| = \sqrt{(v_j \Delta t_k)^2 + \frac{g^2}{4}(\Delta t_k)^4} > 0 \tag{6}$$

### 2.2 无人机位置与投放点的一致性

无人机在$t_{0,k}$时刻到达投放点：
$$\mathbf{P}_k^{drop} = \mathbf{P}_{j(k)}^0 + v_{j(k)} \hat{\mathbf{n}}_{j(k)} \cdot t_{0,k} \tag{7}$$

由于无人机等高度飞行：
$$z_k^{drop} = z_{j(k)}^0 \tag{8}$$

## 三、烟幕云团动态演化模型

### 3.1 烟幕中心运动

起爆后烟幕云团中心以恒定速度下沉。设第$k$枚弹在$t_{1,k}$时刻于$\mathbf{P}_k^{det,0} = (x_k^{det}, y_k^{det}, z_k^{det})$起爆，则$t \in [t_{1,k}, t_{2,k}]$时刻烟幕中心位置为：
$$\mathbf{P}_k^{det}(t) = \left(x_k^{det},\, y_k^{det},\, z_k^{det} - v_{sink}(t - t_{1,k})\right) \tag{9}$$

即：
$$z_k^{det}(t) = z_k^{det,0} - 3(t - t_{1,k}) \tag{10}$$

### 3.2 有效干扰时空域

第$k$枚弹形成的有效干扰区域为时空柱体：
$$\mathcal{D}_k = \left\{(x,y,z,t) : (x-x_k^{det})^2 + (y-y_k^{det})^2 + (z - z_k^{det}(t))^2 \leq R_{eff}^2,\; t \in [t_{1,k}, t_{1,k}+\tau]\right\} \tag{11}$$

## 四、导弹运动学模型

第$m$枚导弹在时刻$t$的位置：
$$\mathbf{P}_{Mm}(t) = \mathbf{P}_{Mm}^0 + v_M \hat{\mathbf{d}}_m \cdot t = \mathbf{P}_{Mm}^0 \left(1 - \frac{v_M t}{L_m}\right) \tag{12}$$

导弹到达假目标（原点）的时间：
$$t_m^{arr} = \frac{L_m}{v_M} \tag{13}$$

具体数值：

- $t_2^{arr} = 19116.22/300 = 63.721$ s

## 五、导弹与干扰球面相交的代数条件

### 5.1 相交判定方程

设第$m$枚导弹在时刻$t$的位置为$\mathbf{P}_{Mm}(t)$，第$k$枚烟幕在同时刻的中心为$\mathbf{P}_k^{det}(t)$。定义相对位置向量：
$$\mathbf{r}_{mk}(t) = \mathbf{P}_{Mm}(t) - \mathbf{P}_k^{det}(t) \tag{14}$$

导弹与烟幕球面相交的条件为：
$$\|\mathbf{r}_{mk}(t)\|^2 = R_{eff}^2 \tag{15}$$

展开得：
$$\|\mathbf{P}_{Mm}^0 + v_M \hat{\mathbf{d}}_m t - \mathbf{P}_k^{det} + (0, 0, v_{sink}(t-t_{1,k}))\|^2 = R_{eff}^2 \tag{16}$$

### 5.2 关于时间$t$的二次方程

定义导弹速度向量$\mathbf{V}_m = v_M \hat{\mathbf{d}}_m = (V_{mx}, V_{my}, V_{mz})$，烟幕中心竖直运动带来的等效速度$\mathbf{V}_k^{sink} = (0, 0, -v_{sink})$。令$\mathbf{V}_{mk}^{rel} = \mathbf{V}_m - \mathbf{V}_k^{sink} = (V_{mx}, V_{my}, V_{mz}+v_{sink})$，并设参考点：
$$\mathbf{Q}_{mk} = \mathbf{P}_{Mm}^0 - \mathbf{P}_k^{det} + (0, 0, v_{sink}t_{1,k}) \tag{17}$$

则方程化为标准二次形式：
$$a_{mk} t^2 + b_{mk} t + c_{mk} = 0 \tag{18}$$

其中系数为：
$$a_{mk} = \|\mathbf{V}_{mk}^{rel}\|^2 = V_{mx}^2 + V_{my}^2 + (V_{mz}+v_{sink})^2 \tag{19}$$
$$b_{mk} = 2(\mathbf{Q}_{mk} \cdot \mathbf{V}_{mk}^{rel}) \tag{20}$$
$$c_{mk} = \|\mathbf{Q}_{mk}\|^2 - R_{eff}^2 \tag{21}$$

### 5.3 判别式与完整情形分析

定义判别式：
$$\Delta_{mk} = b_{mk}^2 - 4a_{mk}c_{mk} \tag{22}$$

**情形一：$\Delta_{mk} < 0$（无交）**

方程无实根，导弹轨迹与烟幕球面无交点。此时：
- 若$c_{mk} > 0$（即$\|\mathbf{Q}_{mk}\| > R_{eff}$）：导弹始终在球面外，无干扰
- 若$c_{mk} < 0$：导弹起始于球面内（物理上需结合时间区间判断）

**情形二：$\Delta_{mk} = 0$（相切）**

唯一交点：
$$t_{mk}^* = -\frac{b_{mk}}{2a_{mk}} \tag{23}$$

需检验$t_{mk}^* \in [t_{1,k}, t_{1,k}+\tau]$且$t_{mk}^* \in [0, t_m^{arr}]$。相切时导弹恰好擦过烟幕边界，有效干扰时间趋于零。

**情形三：$\Delta_{mk} > 0$（贯穿或包含）**

两实根：
$$t_{mk}^{\pm} = \frac{-b_{mk} \pm \sqrt{\Delta_{mk}}}{2a_{mk}} \tag{24}$$

**子情形3a：贯穿（$c_{mk} > 0$，即导弹起始于球面外）**

需$b_{mk} < 0$保证至少一正根。有效进入和离开时刻：
$$t_{mk}^{enter} = t_{mk}^- = \frac{-b_{mk} - \sqrt{\Delta_{mk}}}{2a_{mk}}, \quad t_{mk}^{exit} = t_{mk}^+ = \frac{-b_{mk} + \sqrt{\Delta_{mk}}}{2a_{mk}} \tag{25}$$

导弹在球内时段：$[t_{mk}^{enter}, t_{mk}^{exit}] \cap [t_{1,k}, t_{1,k}+\tau]$

**子情形3b：包含（$c_{mk} < 0$，即导弹起始于球面内）**

此时$t_{mk}^- < 0 < t_{mk}^+$，导弹从球内出发。有效干扰从$\max(0, t_{1,k})$开始，至$\min(t_{mk}^+, t_{1,k}+\tau)$结束。

**子情形3c：起始于边界（$c_{mk} = 0$）**

一根为零，另一根为$t_{mk}^+ = -b_{mk}/a_{mk}$（设$b_{mk} < 0$）。

### 5.4 有效干扰时间计算

综合上述情形，第$k$枚弹对第$m$枚导弹的有效干扰时间区间为：
$$\mathcal{I}_{mk} = [\max(t_{mk}^-, t_{1,k}, 0),\; \min(t_{mk}^+, t_{1,k}+\tau, t_m^{arr})] \tag{26}$$

当且仅当左端点小于右端点时存在有效干扰，持续时长：
$$\Delta t_{mk}^{eff} = \max\left\{0,\; \min(t_{mk}^+, t_{1,k}+\tau, t_m^{arr}) - \max(t_{mk}^-, t_{1,k}, 0)\right\} \tag{27}$$

## 六、视线遮蔽判定条件

### 6.1 几何要素定义

**真目标圆柱体**：以$\mathbf{P}_T = (0, 200, 0)$为下底面圆心，半径$R_{cyl}=7$ m，高$H_{cyl}=10$ m，轴向沿$z$轴。圆柱空间：
$$\mathcal{C} = \left\{(x,y,z) : x^2 + (y-200)^2 \leq R_{cyl}^2,\; z \in [0, H_{cyl}]\right\} \tag{28}$$

**导弹-真目标连线**：时刻$t$时，导弹位置$\mathbf{P}_{Mm}(t)$与真目标中心（或其表面最近点）的连线。

### 6.2 视线遮蔽的严格定义

烟幕对第$m$枚导弹形成有效视线遮蔽，当且仅当同时满足：

**条件C1（距离条件）**：烟幕中心到导弹-真目标连线的距离不超过有效半径
$$d\left(\mathbf{P}_k^{det}(t),\; \overline{\mathbf{P}_{Mm}(t)\mathbf{P}_T}\right) \leq R_{eff} \tag{29}$$

其中点到线段距离：
$$d(\mathbf{P}, \overline{\mathbf{AB}}) = \begin{cases} \|\mathbf{P}-\mathbf{A}\| & \text{if } (\mathbf{P}-\mathbf{A})\cdot(\mathbf{B}-\mathbf{A}) \leq 0 \\ \|\mathbf{P}-\mathbf{B}\| & \text{if } (\mathbf{P}-\mathbf{B})\cdot(\mathbf{A}-\mathbf{B}) \leq 0 \\ \dfrac{\|(\mathbf{P}-\mathbf{A})\times(\mathbf{B}-\mathbf{A})\|}{\|\mathbf{B}-\mathbf{A}\|} & \text{otherwise} \end{cases} \tag{30}$$

**条件C2（位置条件）**：烟幕中心位于导弹与真目标之间（含延长线约束的合理区间）

设参数化直线$\mathbf{L}(s) = \mathbf{P}_{Mm}(t) + s(\mathbf{P}_T - \mathbf{P}_{Mm}(t))$，烟幕中心对应的参数：
$$s_k(t) = \frac{(\mathbf{P}_k^{det}(t) - \mathbf{P}_{Mm}(t)) \cdot (\mathbf{P}_T - \mathbf{P}_{Mm}(t))}{\|\mathbf{P}_T - \mathbf{P}_{Mm}(t)\|^2} \tag{31}$$

要求：
$$s_k(t) \in [\epsilon,\; 1-\epsilon] \tag{32}$$

其中$\epsilon > 0$为微小裕量，确保烟幕不恰好位于导弹或目标位置。

**条件C3（时间重叠）**：$t \in [t_{1,k}, t_{1,k}+\tau] \cap [0, t_m^{arr}]$
