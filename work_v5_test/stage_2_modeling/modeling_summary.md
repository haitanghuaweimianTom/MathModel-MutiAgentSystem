**建模摘要**

**核心变量**  无人机航向角 $\alpha$（rad，决策变量）、干扰弹投放时刻 $t_1$（s，待优化）、有效遮蔽时长 $T_{eff}$（s，核心指标）、烟幕有效遮蔽半径 $r_{eff}=10$（m）、导弹速度 $v_M=300$（m/s）。

**核心公式**  干扰弹起爆点由投放点与自由飞行决定：$P_{burst} = P_{drop} + \vec{v}_{drop}\Delta t_{burst} - \frac{1}{2}g(\Delta t_{burst})^2\hat{z}$，其中 $\Delta t_{burst}=3.6$ s；烟幕中心时变位置受下沉影响：$P_{smoke}(t) = P_{burst} - v_{sink}(t-t_2)\hat{z}$，$t \geq t_2$。有效遮蔽时长定义为 $T_{eff} = t_{eff,end} - t_{eff,start}$，需满足烟幕云团同时覆盖真目标与导弹视线的几何约束。

**求解策略**  以最大化 $T_{eff}$ 为目标，联合优化投放时刻 $t_1$ 与航向角 $\alpha$，通过运动学递推判定烟幕-目标-导弹三点共线遮蔽条件。

**与子问题的映射**  子问题1（单弹干扰单导弹）：确定最优 $(t_1^*, \alpha^*)$ 使 $T_{eff}$ 最大化；子问题2（单弹干扰多导弹）：扩展为遮蔽时段需覆盖多导弹末制导关键窗口的交集约束；子问题3（多弹协同）：引入多弹投放时刻序列 $\{t_{1,k}\}$ 与空间布点，实现烟幕接续或并置覆盖以延长总有效遮蔽时长。
