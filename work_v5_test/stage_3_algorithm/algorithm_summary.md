**算法名称**：导弹-圆柱体碰撞检测算法

**输入数据**：导弹索引idx（0-2）、时间t、圆柱体参数（中心c、半径R、高度H）、采样线段端点p1/p2

**核心步骤**：
1. **导弹轨迹生成**：基于目标坐标计算初始距离与单位方向向量，按匀速300m/s直线运动方程P(t)=P0+v·t·dir推导实时三维坐标
2. **线段-圆柱体求交**：将三维问题降维至XY平面，通过二次方程判别式disc=b²-4ac判定射线与底面圆相交性，求解进出参数t1/t2
3. **碰撞有效性判定**：约束t_enter∈[0,1]且Z坐标∈[0,H]，排除线段反向延长或高度越界情形

**输出格式**：布尔值（True/False），表示该时刻导弹是否与圆柱体相交
