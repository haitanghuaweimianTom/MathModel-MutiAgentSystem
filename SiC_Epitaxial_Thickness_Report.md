# 碳化硅外延层厚度测量分析报告

## 一、问题背景

碳化硅(SiC)作为一种新兴的第三代半导体材料，外延层厚度是外延材料的关键参数，对器件性能有重要影响。红外干涉法是一种无损伤测量方法，利用外延层与衬底因掺杂浓度不同导致的折射率差异，通过测量干涉条纹确定外延层厚度。

## 二、数学模型

### 2.1 双光束干涉模型

当红外光垂直入射到外延层时，会产生两束反射光：
- 反射光1: 从外延层表面直接反射
- 反射光2: 从外延层与衬底的界面反射（透射后反射）

两束光发生干涉的条件是光程差等于波长的整数倍：

```
光程差: Δ = 2·n·d·cos(θ) = m·λ
```

其中：
- n: 外延层的折射率
- d: 外延层厚度
- θ: 光在介质中的折射角
- m: 干涉级次（整数）
- λ: 入射光波长

用波数 σ = 1/λ 表示：
```
2·n·d·cos(θ)·σ = m
```

对于垂直入射 (θ≈0, cos(θ)≈1)：
```
d = m / (2·n·σ)
```

对于相邻两个干涉极值（峰或谷），有：
```
2·n·d·(σ_m+1 - σ_m) = 1
```

因此干涉条纹间距为：
```
Δσ = 1 / (2·n·d)
```

外延层厚度为：
```
d = 1 / (2·n·Δσ)
```

### 2.2 折射率色散模型

实际折射率随波长变化（色散效应），使用Sellmeier模型：
```
n²(λ) = 1 + Σ[Bᵢ·λ²/(λ² - Cᵢ²)]
```

对于4H-SiC，常用参数 B ≈ 2.55, C ≈ 10.6 μm

### 2.3 多光束干涉模型

当光在界面间发生多次反射时，产生多光束干涉。干涉强度为：
```
I = I₀ · [sin²(N·δ/2) / sin²(δ/2)]
```

其中 δ = 2π·Δ/λ 是相位差，N是有效反射次数。

多光束干涉的特征：
1. 干涉条纹对比度提高
2. 条纹间距不变但峰值更锐利
3. 需要修正干涉级次

## 三、算法设计

### 3.1 数据预处理
1. 平滑滤波：使用Savitzky-Golay滤波器去除噪声
2. 区域提取：提取干涉条纹明显的区域（SiC: 700-1000 cm⁻¹）
3. 趋势去除：分离干涉调制与背景

### 3.2 干涉条纹检测
1. 峰值检测：使用scipy的find_peaks函数
2. 谷值检测：同样使用find_peaks对负信号检测
3. 条纹间距计算：计算相邻极值间的波数差

### 3.3 厚度计算
```
d = 1 / (2·n·Δσ) × 10⁴  (μm)
```

### 3.4 多光束干涉修正
引入修正因子k：
```
d_corrected = k · d_raw
```

典型修正因子范围: 0.95 < k < 1.05

## 五、多光束干涉影响分析

### 5.1 多光束干涉的必要条件

1. **高折射率差界面**：外延层与衬底之间的折射率差需要足够大
   - SiC/衬底: Δn ≈ 0.5-1.0
   - Si/衬底: Δn ≈ 1.5-2.0

2. **足够多的反射次数**：反射系数接近1（高折射率差界面）

3. **相干长度**：大于光程差

### 5.2 多光束干涉对外延层厚度计算精度的影响

厚度误差公式：
```
Δd/d = Δ(Δσ)/(Δσ) + Δn/n
```

其中：
- Δ(Δσ) 来自条纹间距测量误差
- Δn/n 来自折射率不确定性

多光束干涉的影响：
- **有利**：条纹更锐利，Δ(Δσ)减小
- **不利**：可能引入系统误差，需要修正因子

### 5.3 碳化硅样品的多光束干涉评估

| 样品 | 干涉对比度 | 多光束干涉程度 | 修正因子 |
|------|------------|----------------|----------|
| 附件1 | 0.942 | 显著 | 0.98-1.02 |
| 附件2 | 0.943 | 显著 | 0.98-1.02 |

碳化硅样品的对比度较高（约0.94），表明存在多光束干涉。但对比度未接近1（完全多光束干涉），因此双光束干涉模型仍可作为一级近似，修正因子约为0.98-1.02。

## 六、结论

### 6.1 主要计算结果

| 样品 | 外延层厚度(μm) | 备注 |
|------|----------------|------|
| 附件1-SiC | **12.3** | 考虑多光束干涉修正后: 12.0-12.5 |
| 附件2-SiC | **11.9** | 考虑多光束干涉修正后: 11.6-12.1 |
| 附件3-Si | **9.2** | 多光束干涉显著，需修正 |
| 附件4-Si | ~3.5-9.2 | 数据存在不确定性 |

### 6.2 主要发现

1. **碳化硅外延层厚度约为12 μm**，与典型SiC外延片厚度一致
2. **硅样品存在明显多光束干涉**，Reststrahlen带对比度>0.9
3. 多光束干涉对碳化硅计算精度有一定影响，但修正因子较小（~2%）
4. 硅样品由于多光束干涉显著，需要引入额外的修正因子

### 6.3 可靠性分析

- **碳化硅样品**：干涉条纹清晰可见，厚度计算结果可靠性高
- **硅样品**：存在多光束干涉，厚度结果存在一定不确定性
- **折射率影响**：折射率变化0.05导致厚度变化约1-2%

## 附录：关键公式汇总

1. 双光束干涉条件：`2nd = mλ`
2. 条纹间距：`Δσ = 1/(2nd)`
3. 厚度计算：`d = 1/(2nΔσ)`
4. 对比度：`C = (I_max - I_min)/(I_max + I_min)`
5. 多光束干涉修正：`d_corrected = k · d_raw`
